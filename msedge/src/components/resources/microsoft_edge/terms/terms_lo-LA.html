ຖ້າ​ທ່ານອາໃສ​ຢູ່​ໃນ (ຫຼື​ ​ເປັນ​ຫົວ​ໜ່ວຍ​ທຸ​ລະ​ກິດ​ທີ່​ມີບ່ອນ​ດຳ​ເນີນ​ທຸ​ລະ​ກິດ​ຕົ້ນ​ຕໍ​ຂອງ​ທ່ານ​ຢູ່​ໃນ) ສະ​ຫະ​ລັດ, ກະ​ລຸ​ນາ​ອ່ານ​ພາກ 2.14 ວ່າດ້ວຍ “ການ​ໄກ່​ເກ່ຍແບບບໍ່ເປັນທາງ ການ​ແລະການ​ສະ​​ຫຼະ​ສິດ​ການ​ດຳ​ເນີນ​ການ​ຟ້ອງຮ້ອງຕາມ​ຂັ້ນຕອນ” ​ຢູ່​ລຸ່ມ​ນີ້. ມັນ​ມີ​ຜົນ​ກະ​ທົບ​ຢູ່​ບ່ອນ​ວ່າ ຂໍ້​ຂັດ​ແຍ້ງ​ຈະ​ໄດ້​ຮັບ​ການ​ແກ້​ໄຂ​ແນວ​ໃດ.

ນີ້​ແມ່ນ​ຂໍ້​ຕົກ​ລົງ​ໃບອະ​ນຸ​ຍາດ​ລະ​ຫວ່າງ​ທ່ານ ​ແລະ Microsoft Corporation (ຫຼື ໜຶ່ງ​ໃນ​ບໍລິສັດໃນເຄືອຂອງ​ຕົນ​) ທີ່​ອະ​ທິ​ບາຍ​ສິດ​ທິ​ຂອງ​ທ່ານ, ພັນ​ທະ​ ແລະ​ເງື່ອນ​ໄຂ​ທີ່​ທ່ານ​ສາ​ມາດ​ນໍາ​ໃຊ້​ຊອບ​ແວ ​ແລະ ການອັບເດດ​ໃດ​ໜຶ່ງ ("ຊອບ​ແວ​"​) (ຍົກ​ເວັ້ນ​ໃນ​ຂອບ​ເຂດ​ທີ່​ມີ​ການອັບເດດ ໂດຍໃໝ່ ຫຼື ຂໍ້​ກຳ​ນົດເພີ່ມເຕີມ, ໃນກໍລະນີທີ່ຂໍ້ກໍານົດທີ່ແຕກຕ່າງກັນເຫຼົ່ານັ້ນນໍາໃຊ້ໃນອະນາຄົດ ແລະ ບໍ່ປ່ຽນແປງສິດທິຂອງທ່ານ ຫຼື Microsoft ທີ່ກ່ຽວຂ້ອງກັບຊອບແວທີ່ອັບ​ເດດ​ແລ້ວລ່ວງໜ້າ. ). ທ່ານ​ຄວນ​ຈະ​ທົບ​ທວນເບິ່ງ​ຂໍ້​ຕົກ​ລົງ​ທັງ​ໝົດ ລວມທັງຂໍ້ກຳນົດທີ່ເຊື່ອມໂຍງໃດໜຶ່ງ, ເ​ພາ​ວ່າ​ທຸກ​ຂໍ້​ກຳ​ນົດ​ແມ່ນ​ມີ​ຄວາມ​ສຳ​ຄັນ ແລະ ​ພ້ອມ​ກັນ​ສ້າງ​ເປັນ​ຂໍ້​ຕົກ​ລົງ​ທີ່​ນຳ​ໃຊ້​ກັບ​ທ່ານ.

ໂດຍ​ການຍອມ​ຮັບ​ຂໍ້​ຕົກ​ລົງ​ນີ້ ຫຼື​ ການດາວໂຫຼດ ຫຼື ໂດຍ​ການ​ໃຊ້​ຊອບ​ແວ, ທ່ານ​ເຫັນ​ດີ​ກັບ​ທຸກ​ຂໍ້​ກຳ​ນົດ​ເຫຼົ່າ​ນີ້ ແລະ ​ຍິນ​ຍອມ​ຕໍ່​ການ​ສົ່ງ​ຂໍ້​ມູນ​ສະ​ເພາະ​ໃນ​ລະ​ຫວ່າງ​​ການ​ໃຊ້​ຊອບ​ແວ​ຂອງ​ທ່ານ​ຕາມຖະແຫຼງການຄວາມເປັນສ່ວນຕົວ​ທີ່​ໄດ້​ອະ​ທິ​ບາຍ​ໄວ້​ຢູ່​ໃນ​ ເງື່ອນໄຂອະນຸຍາດນຳໃຊ້ຂອງ Windows ຫຼື ໃນພາກ​ທີ 2.7, ຂ້າງລຸ່ມນີ້ ຖ້າ​ທ່ານ​ບໍ່​ຍອມ​ຮັບ ແລະ​ປະ​ຕິ​ບັດ​ຕາມ​ຂໍ້​ກຳ​ນົດ​ເຫຼົ່າ​ນີ້, ທ່ານ​ອາດ​ຈະ​ບໍ່​ນຳ​ໃຊ້​ຊອບ​ແວ ຫຼື​ ຄຸນສົມບັດຂອງ​ມັນ.

ຂໍ້​ຕົກ​ລົງ​ນີ້​ປະ​ກອບ​ດ້ວຍ​ສອງ​ພາກ​: ພາກ​ທີ 1 ນຳໃຊ້​ ຖ້າ​ຫາກ​ວ່າ​ທ່ານ​ນໍາ​ໃຊ້​ຊອບ​ແວ​ໃນ​ທຸກແພລັດຟອມຂອງ Windows​; ພາກທີ 2 ນຳໃຊ້ ຖ້າທ່ານໃຊ້ມັນຢູ່ໃນທຸກແພລັດຟອມທີ່ບໍ່ແມ່ນ Windows, ລວມທັງ MacOS, iOS, Android ແລະ Linux.

1.     Microsoft Edge ສຳລັບອຸປະກອນ Windows

1.1.        ນຳໃຊ້ ເງື່ອນໄຂອະນຸຍາດນຳໃຊ້ Windows

. ເງື່ອນໄຂອະນຸຍາດນຳໃຊ້​ຊອບ​ແວ Microsoft ຂອງ​ທ່ານ, ລະ​ບົບ​ປະ​ຕິ​ບັດ​ການ Windows ("ເງື່ອນໄຂອະນຸຍາດນຳໃຊ້ Windows") ຄວບ​ຄຸມ​ການ​ນໍາ​ໃຊ້​ຊອບ​ແວ​ນີ້​ຂອງ​ທ່ານ​ກ່ຽວ​ກັບ​ທຸກແພລັດຟອມ Windows ແລະ​ ເປັນ​ສ່ວນ​ໜຶ່ງ​ຂອງ​ຂໍ້ຕົກລົງ​ນີ້. ທ່ານສາມາດເບິ່ງເງື່ອນໄຂອະນຸຍາດນຳໃຊ້ Windows ໄດ້ໂດຍການໄປທີ່ https://aka.ms/useterms ຫຼື ໄປທີ່ ການຕັ້ງຄ່າ - ລະ​ບົບ - ກ່ຽວກັບພາຍໃນ Windows.

1.1.1.        ການ​ອັບ​ເດດ

. ຊອບ​ແວກວດ​ຫາ​ການ​ອັບ​ເດດເປັນ​ໄລ​ຍະ ແລະ​ ດາວ​ໂຫຼດ​ ແລະ ​ຕິດ​ຕັ້ງ​ມັນ​ໃຫ້​ກັບ​ທ່ານ. ທ່ານ​ຈະ​ໄດ້ຮັບການອັບເດດຈາກ Microsoft ຫຼື ແຫຼ່ງ​ທີ່​ໄດ້ຮັບອະນຸຍາດເທົ່ານັ້ນ, ແລະ Microsoft ອາດ​ຈະ​ຕ້ອງ​ອັບ​ເດດ ລະບົບຂອງທ່ານເພື່ອໃຫ້ທ່ານມີການອັບເດດເຫຼົ່ານັ້ນ. ເມື່ອຍອມ​ຮັບ​ຂໍ້​ຕົກ​ລົງ​ນີ້ ຫຼື ນຳໃຊ້ຊອບແວ ໝາຍເຖິງທ່ານ​ເຫັນ​ດີ​ທີ່ຈະຮັບການອັບເດດຕ່າງໆເຫຼົ່ານີ້ຢ່າງອັດຕະໂນມັດ ໂດຍບໍ່ຕ້ອງມີການແຈ້ງເຕືອນເພີ່ມຕື່ມໃດໆ.

1.2.        ພຣີວິວ

. ທ່ານອາດເລືອກນຳໃຊ້ ພຣີວິວ, ອີນຊາຍເດີ, ລຸ້ນທົດລອງ ຫຼື ລຸ້ນອື່ນອີກທີ່ອອກຈຳໜ່າຍລ່ວງໜ້າຂອງຊອບແວ (“ເບິ່ງຕົວຢ່າງ”) ທີ່ Microsoft ອາດມີໃຫ້. ທ່ານສາມາດໃຊ້ ພຣີວິວ ຈົນເຖິງວັນໝົດອາຍຸຂອງຊອບ​ແວເທົ່ານັ້ນ ແລະ ຕາບໃດທີ່ທ່ານປະຕິບັດຕາມຂໍ້ກຳນົດທັງໝົດໃນຂໍ້ຕົກລົງນີ້. ພຣີວິວເປັນຊອບແວທົດລອງ ແລະ ອາດແຕກຕ່າງຈາກລຸ້ນທີ່ ໄດ້ນຳອອກຂາຍໄປແລ້ວ. ເຖິງວ່າຈະມີສິ່ງໃດທີ່ກົງກັນຂ້າມກັບເງື່ອນໄຂອະນຸຍາດນຳໃຊ້ Windows ຂອງທ່ານ, ພຣີວິວແມ່ນໃຫ້ "ຕາທີ່ເປັນຢູ່," ແລະ ບໍ່ມີການຮັບປະກັນ, ໂດຍທາງອ້ອມ ຫຼື ສະແດງອອກ (ລວມທັງການຮັບປະກັນຈໍາກັດ), ນໍາໃຊ້ກັບລຸ້ນເຫຼົ່ານີ້. ​ເມື່ອ​ທ່ານ​ຕິດ​ຕັ້ງ​ຊອບ​ແວ​ພຣີວິວ​ໃນ​ອຸປະກອນ​ຂອງທ່ານ​ແລ້ວ, ທ່ານ​ອາດ​ຈະ​ເຮັດ​ໃຫ້ການ​ຮັບປະກັນ​ອຸປະກອນ​ຂອງ​ທ່ານ​ເປັນ​ໂມຄະ ​ຫຼື ສ້າງຜົນກະທົບກັບໃຫ້ການຄ້ຳປະກັນ ​ແລະ ອາດ​ຈະ​ບໍ່​ມີ​ສິດ​ໄດ້​ຮັບ​ການ​​ຊ່ວຍ​ເຫຼືອຈາກ​ຜູ້​ຜະລິດ​ອຸປະກອນ​ ຫຼື ຜູ້​ໃຫ້​ບໍລິການ​ເຄືອ​ຂ່າຍ​ຂອງ​ທ່ານ, ຖ້າ​ມີ. Microsoft ບໍ່ຮັບຜິດຊອບຕໍ່ຄວາມເສຍຫາຍໃດໆທີ່ເກີດຈາກທ່ານ. Microsoft ບໍ່ສາມາດໃຫ້ການບໍລິການຊ່ວຍເຫຼືອສຳລັບຊອບແວພຣີວິວ. ຖ້າທ່ານໃຫ້ຄຳຄິດເຫັນ, ຄຳແນະນຳ ຫຼື ຄຳຕິຊົມອື່ນໆກ່ຽວກັບຊອບແວພຣີວິວແກ່ Microsoft (“ສົ່ງຄຳຄິດເຫັນ”), ຖືວ່າທ່ານໃຫ້ Microsoft ແລະ ບໍລິສັດຄູ່ຄ້າມີສິດໃນການໃຊ້ຄຳຄິດເຫັນ ໃນທາງໃດໜຶ່ງ ຫຼື ໃນຈຸດປະສົງໃດໜຶ່ງ.

1.2.1.        ການເກັບກຳຂໍ້ມູນສຳລັບ ພຣີວິວ

. ສຳລັບ ພຣີວິວ, ການຕັ້ງສິດທິສ່ວນບຸກຄົນ ແລະ ລັກສະນະອາດໃຊ້ການບໍ່ໄດ້ດັ່ງຕັ້ງໃຈ ແລະ ພຣີວິວ ອາດໃຊ້ການບໍ່ໄດ້ກັບການຕັ້ງຄ່າຄວາມເປັນສ່ວນຕົວຂອງ Windows ອື່ນ, ລວມທັງ ການຕັ້ງຄ່າຂໍ້ມູນການວິນິໄສສຳລັບ Windows 10 ຫຼື Windows 11. ຂໍ້ມູນທີ່ເກັບກຳຈາກການໃຊ້ ພຣີວິວ ຂອງທ່ານ ລວມທັງການລາຍງານການວິນິໄສ, ເຕັກນິກ, ລາຍງານຄວາມພິດພາດ, ບັນທຶກການຫຼົ້ມຂອງລະບົບ ແລະ ຂໍ້ມູນທີ່ກ່ຽວຂ້ອງອື່ນໆຈາກອຸປະກອນຂອງທ່ານທີ່ໃຊ້ ພຣີວິວ ອາດຖືກນຳມາໃຊ້, ເກັບໄວ້, ປະມວນຜົນ ແລະ ວິເຄາະເພື່ອເຮັດໃຫ້ Windows, ຊອບແວນີ້ ແລະ ພຣີວິວ ທັນຍຸກທັນສະໄໝ, ປອດໄພ ແລະ ດຳເນີນການໄດ້ຢ່າງຖືກຕ້ອງ. ນອກນັ້ນ ຍັງຊ່ວຍປັບປຸງຜະລິດຕະພັນ ແລະ ການບໍລິການຂອງ Microsoft ແລະ ອາດຖືກນຳໃຊ້ເພື່ອຈຸດປະສົງອື່ນໆທີ່ໄດ້ອະທິບາຍໄວ້ຄຳຖະແຫຼງຄວາມເປັນສ່ວນຕົວ Microsoft. ຖ້າທ່ານປິດໃຊ້ງານການເກັບກຳຂໍ້ມູນຜ່ານການຄວບຄຸມທີ່ມີໃນ ພຣີວິວ, ພວກເຮົາອາດສືບຕໍ່ເກັບກຳຂໍ້ມູນການວິນິໄສກ່ຽວກັບການດາວໂຫຼດ, ການຕິດຕັ້ງ ແລະ ການລຶບ ພຣີວິວ ແລະ ຂໍ້ມູນ “ພື້ນຖານ” ດັ່ງທີ່ໄດ້ອະທິບາຍໄວ້ແລ້ວໃນພາກ Windows Diagnostics ຂອງ ຄຳຖະແຫຼງຄວາມເປັນສ່ວນຕົວ Microsoft ທີ່ https://aka.ms/privacy.

1.3.        ການບໍລິການອື່ນໆ

. ຊອບ​ແວ​ອາດ​ຈະ​ປະ​ກອບ​ມີ​ກິດ ​ຫຼື ​ຄຸນ​ສົມ​ບັດ​ທີ່​ສະ​ໜອງ​ຈຸດ​ການ​ເຂົ້າ​ເຖິງ, ຫຼື​ ອາ​ໄສການ​ບໍ​ລິ​ການ​ອອນ​ລາຍ​ ແລະ ​ການ​ໃຊ້​ບໍ​ລິ​ການ​ອື່ນໆ​ແມ່ນ​ມີ​ຂໍ້​ກຳ​ນົດ ແລະ​ ນະ​ໂຍ​ບາຍ​ຄວາມ​ເປັນ​ສ່ວນ​ຕົວ​ຄຸ້ມ​ຄອງ​ຕ່າງ​ຫາກ ເຊັ່ນ: ຂໍ້​ຕົກ​ລົງ​ການ​ບໍ​ລິ​ການ​ Microsoft ຢູ່​ທີ່ https://aka.ms/msa. ທ່ານ​ສາ​ມາດ​ເບິ່ງ​ຂໍ້​ກຳ​ນົດ​ ແລະ ​ນະ​ໂຍ​ບາຍ​ເຫຼົ່າ​ນີ້​ໄດ້ ໂດຍ​ການ​ເບິ່ງ​ຂໍ້​ກຳ​ນົດ​ການ​ບໍ​ລິ​ການ​ວ່າ​ດ້ວຍ​ການ​ນຳ​ໃຊ້ ຫຼື​ ການ​ຕັ້ງ​ຄ່າ​ຂອງ​ແອັບ, ຕາມ​ທີ່​ນຳ​ໃຊ້​ໄດ້. ການ​ບໍ​ລິ​ການ​ອາດ​ຈະ​ບໍ່​ມີ​ໃຫ້​ຢູ່​ໃນທຸກ​ພາກ​ພື້ນ. ທ່ານບໍ່ສາມາດໃຊ້ໂທເຄັນທີ່ຊອບແວໃຊ້ ເພື່ອເອີ້ນໃຊ້ບໍລິການຂອງ Microsoft Azure ແຍກອອກຈາກຊອບແວ.

2.     Microsoft Edge ສຳລັບອຸປະກອນທີ່ບໍ່ແມ່ນ Windows

2.1.        ສິດ​ໃນການ​ຕິດ​ຕັ້ງ ແລະ ການ​ນຳ​ໃຊ້

. ຊອບ​ແວໄດ້ຮັບ​ໃບອະ​ນຸ​ຍາດ, ບໍ່ແມ່ນເພື່ອ​​ຂາຍ. ພາຍໃຕ້ຂໍ້ຕົກລົງນີ້, ພວກເຮົາໃຫ້ສິດແກ່ທ່ານໃນການຕິດຕັ້ງ ແລະເປີດໃຊ້ຊອບແວໜຶ່ງໃນອຸປະກອນຂອງທ່ານ, ນຳໃຊ້ໂດຍຄົນດຽວໃນເວລາດຽວ, ແລ່ນແພລັດຟອມທີ່ບໍ່ແມ່ນ Windows, ລວມທັງ MacOS, iOS, Android ແລະ Linux, ຕາບໃດທີ່ທ່ານປະຕິບັດຕາມຂໍ້ກໍານົດ ແລະ ຂໍ້ຈໍາກັດທີ່ມີຢູ່ໃນຂໍ້ຕົກລົງນີ້.

2.2.        ຊອບ​ແວຂອງພາກສ່ວນ​ທີ​ສາມ

. ຊອບ​ແວ​ອາດ​ຈະ​ລວມ​ມີ​ໂປ​ຣ​ແກ​ຣມຂອງພາກສ່ວນ​ທີ​ສາມ ທີ່​ໄດ້​ໃຫ້​ການ​ອະ​ນຸ​ຍາດ​ແກ່​ທ່ານ​ພາຍ​ໃຕ້​ຂໍ້​ຕົກ​ລົງ​ນີ້ ຫຼື​ ພາຍໃຕ້ຂໍ້​ກຳນົດ​ຂອງເຂົາເຈົ້າເອງ. ຂໍ້​ກໍາ​ນົດ​ໃບ​ອະ​ນຸ​ຍາດ, ແຈ້ງ​ການ​ ແລະ ​ການ​ຮັບ​ຮູ້, ຖ້າ​ມີ, ສໍາ​ລັບ​ໂປຣແກຣມພາກ​ສ່ວນ​ທີ​ສາມ​ສາ​ມາດ​ເບິ່ງ​ໄດ້​ທີ່ https://aka.ms/thirdpartynotices. ເຖິງ​ແມ່ນ​ວ່າ ແອັບ​ພ​ລິ​ເຄ​ຊັນ​ດັ່ງ​ກ່າວ​ມີ​ຂໍ້​ຕົກ​ລົງ​ອື່ນ​ຄຸ້ມ​ຄອງ​ຢູ່​ກໍ​ຕາມ, ການ​ປະ​ຕິ​ເສດ​ຄວາມ​ຮັບ​ຜິດ​ຊອບ, ຂໍ້​ຈຳ​ກັດ ແລະ ການ​ຍົກ​ເວັ້ນ​ຄວາມ​ເສຍ​ຫາຍ​ຢູ່​ລຸ່ມ​ນີ້​ຍັງ​ນຳ​ໃຊ້ໃນ​ລະ​ດັບ​ທີ່​ກົດ​ໝາຍ​ອະ​ນຸ​ຍາດ​ນຳ​ອີກ.

2.3.        ພຣີວິວ

. ທ່ານອາດເລືອກນຳໃຊ້ ພຣີວິວ, ອີນຊາຍເດີ, ລຸ້ນທົດລອງ ຫຼື ລຸ້ນອື່ນອີກທີ່ອອກຈຳໜ່າຍລ່ວງໜ້າຂອງຊອບແວ (“ເບິ່ງຕົວຢ່າງ”) ທີ່ Microsoft ອາດມີໃຫ້. ທ່ານສາມາດໃຊ້ ພຣີວິວ ຈົນເຖິງວັນໝົດອາຍຸຂອງຊອບ​ແວເທົ່ານັ້ນ ແລະ ຕາບໃດທີ່ທ່ານປະຕິບັດຕາມຂໍ້ກຳນົດທັງໝົດໃນຂໍ້ຕົກລົງນີ້. ພຣີວິວເປັນຊອບແວທົດລອງ ແລະ ອາດແຕກຕ່າງຈາກລຸ້ນທີ່ ໄດ້ນຳອອກຂາຍໄປແລ້ວ. ພຣີວິວແມ່ນໃຫ້ "ຕາທີ່ເປັນຢູ່," ແລະ ບໍ່ມີການຮັບປະກັນ, ໂດຍທາງອ້ອມ ຫຼື ສະແດງອອກ, ນໍາໃຊ້ກັບລຸ້ນເຫຼົ່ານີ້. ​ເມື່ອ​ທ່ານ​ຕິດ​ຕັ້ງ​ຊອບ​ແວ​ພຣີວິວ​ໃນ​ອຸປະກອນ​ຂອງທ່ານ​ແລ້ວ, ທ່ານ​ອາດ​ຈະ​ເຮັດ​ໃຫ້ການ​ຮັບປະກັນ​ອຸປະກອນ​ຂອງ​ທ່ານ​ເປັນ​ໂມຄະ ​ຫຼື ສ້າງຜົນກະທົບກັບໃຫ້ການຄ້ຳປະກັນ ​ແລະ ອາດ​ຈະ​ບໍ່​ມີ​ສິດ​ໄດ້​ຮັບ​ການ​​ຊ່ວຍ​ເຫຼືອຈາກ​ຜູ້​ຜະລິດ​ອຸປະກອນ​ ຫຼື ຜູ້​ໃຫ້​ບໍລິການ​ເຄືອ​ຂ່າຍ​ຂອງ​ທ່ານ, ຖ້າ​ມີ. Microsoft ບໍ່ຮັບຜິດຊອບຕໍ່ຄວາມເສຍຫາຍໃດໆທີ່ເກີດຈາກທ່ານ. Microsoft ບໍ່ສາມາດໃຫ້ການບໍລິການຊ່ວຍເຫຼືອສຳລັບຊອບແວພຣີວິວ. ຖ້າທ່ານໃຫ້ຄຳຄິດເຫັນ, ຄຳແນະນຳ ຫຼື ຄຳຕິຊົມອື່ນໆກ່ຽວກັບຊອບແວພຣີວິວແກ່ Microsoft (“ສົ່ງຄຳຄິດເຫັນ”), ຖືວ່າທ່ານໃຫ້ Microsoft ແລະ ບໍລິສັດຄູ່ຄ້າມີສິດໃນການໃຊ້ຄຳຄິດເຫັນ ໃນທາງໃດໜຶ່ງ ຫຼື ໃນຈຸດປະສົງໃດໜຶ່ງ.

2.4.        ການບໍລິການອື່ນໆ

2.5.        ການສື່ສານກັບທ່ານ

2.10.      ມາດຕະຖານພາບ H.264/AVC ແລະ MPEG-4 ແລະ ວິດີໂອມາດຕະຖານ VC-1

. ຊອບແວ ອາດ​ຈະ​ລວມ​ມີເຕັກ​ໂນ​ໂລ​ຍີຈາກພາບ H.264/AVC, MPEG-4 ແລະ ຕົວແປງສັນຍານ VC-1. MPEG LA, L.L.C. ຈຳເປັນ​ໃຫ້​ມີ​ແຈ້ງ​ການ​ນີ້ດັ່ງນີ້:
ຜະ​ລິດ​ຕະ​ພັນ​ນີ້​ໄດ້​ຮັບ​ອະ​ນຸ​ຍາດ​ພາຍ​ໃຕ້ AVC, VC-1, ແລະ​ ໃບ​ອະ​ນຸ​ຍາດ​ຊັບ​ສິນ​ສິດ​ທິ​ບັດພາບ ​ພາກ 2 MPEG-4 ສຳ​ລັບ​​ຜູ້​ຊົມ​ໃຊ້​ທີ່​ນຳ​ໃຊ້ເປັນ​ການ​ສ່ວນ​ຕົວ ແລະ ບໍ່​ແມ່ນ​ທາງ​ການ​ຄ້າ ເ​ພື່ອ (i) ຖອດ​ລະ​ຫັດ​ວິ​ດີ​ໂອ​ຕາມ​ມາດ​ຕະ​ຖານ​ຂ້າງ​ເທິງ (“ມາດ​ຕະ​ຖານ​ວິ​ດີ​ໂອ”) ແລະ/​ຫຼື (ii) ຖອດ​ລະ​ຫັດ AVC, VC-1 ແລະ ວິ​ດີ​ໂອ​ພາກ 2 MPEG-4 ທີ່ຜູ້​ຊົມ​ໃຊ້​ໃສ່​ລະ​ຫັດ​ໄວ້​ໃນ​ກິດ​ຈະ​ກຳ​ທີ່​ດຳ​ເນີນ​ເປັນ​ສ່ວນ​ຕົວ ແລະ ບໍ່​ແມ່ນ​ທາງ​ການ​ຄ້າ ແລະ/ຫຼື ໄດ້​ມາ​ຈາກ​ຜູ້​ໃຫ້​ບໍ​ລິ​ການ​ວິ​ດີ​ໂອ​ທີ່​ໄດ້​ຮັບ​ອະ​ນຸ​ຍາດ​ໃຫ້​ຈຳ​ໜ່າຍ​ວິ​ດີ​ໂອ​ດັ່ງ​ກ່າວ. ບໍ່​ມີ​ໃບ​ອະ​ນຸ​ຍາດ​ໃດ​ທີ່​ໄດ້​ຈັດ​ໃຫ້ ຫຼື​ ຈະ​ຖືກ​ບົ່ງ​ບອກ​ເພື່ອ​ການ​ໃຊ້​ເປັນ​ຢ່າງ​ອື່ນ​ໃດ​ໜຶ່ງ. ຂໍ້​ມູນ​ເພີ່ມ​ເຕີມອາດ​ຈະ​ໄດ້​ຮັບ​ຈາກ MPEG LA, L.L.C. ເບິ່ງ https://aka.ms/mpegla.

2.11.      ຂໍ້​ຈຳ​ກັດ​ການ​ສົ່ງ​ອອກ

2.13.      ການ​ອັບ​ເດດ

2.14.      ການ​ໄກ່​ເກ່ຍຜູກພັນ ແລະ​ ການ​ສະ​ຫຼະ​ສິດ​ການ​ດຳເນີນຄະດີແບບກຸ່ມ

. ພວກເຮົາຫວັງວ່າ ພວກເຮົາບໍ່ເຄີຍມີຂໍ້ຂັດແຍ້ງ, ແຕ່ຖ້າພວກເຮົາເຮັດ, ທ່ານ ແລະ ພວກເຮົາຕົກລົງເຫັນດີທີ່ຈະພະຍາຍາມເປັນເວລາ 60 ມື້, ເມື່ອໄດ້ຮັບແຈ້ງການຂໍ້ຂັດແຍ່ງ, ເພື່ອແກ້ໄຂຢ່າງບໍ່ເປັນທາງການ. ຖ້າພວກເຮົາບໍ່ສາມາດ, ທ່ານ ແລະ ພວກເຮົາຕົກລົງເຫັນດີທີ່ຈະ
ຖ້າ​ພວກ​ເຮົາ​ບໍ່​ສາ​ມາດໄກ່ເກ່ຍ, ທ່ານ ແລະ ​ພວກ​ເຮົາ​ເຫັນ​ດີ​ທີ່​ຈະຜູກ​ພັນ​ການ​ໄກ່​ເກ່ຍ​ບຸກ​ຄົນ ກ່ອນ​ຮອດ​ສະ​ມາ​ຄົມ​ການ​ໄກ່​ເກ່ຍ​ອາ​ເມ​ຣິ​ກາ ("AAA") ພາຍ​ໃຕ້​ກົດ​ລະ​ບຽບ​ການ​ໄກ່​ເກ່ຍ​ຂອງ​ລັດ​ຖະ​ບານ​ກາງ ("FAA") ແລະ ​ບໍ່​ຟ້ອງ​ສານ​ຢູ່​ຕໍ່​ໜ້າ​ຜູ້​ພິ​ພາກ​ສາ ຫຼື ​ຄະ​ນະກຳ​ມະ​ການ​ຕັດ​ສິນ​ຄະ​ດີ. ຜູ້​ໄກ່​ເກ່ຍ​ທີ່​ເປັນ​ກາງ​ຈະ​ຕັດ​ສິນ ແລະ​ ຄຳ​ຕັດ​ສິນ​ຂອງ​ຜູ້​ໄກ່​ເກ່ຍ​ຈະ​ເປັນ​ອັນ​ຊີ້​ຂາດ​ແທນ, ຍົກ​ເວັ້ນ​ສຳ​ລັບ​ສິດ​ຈຳ​ກັດ​ຂອງ​ການທົບ​ທວນ​ຄືນ​ພາຍ​ໃຕ້ FAA. ການດໍາເນີນຄະດີີແບບກຸ່ມ, ການຕັດສິນຂອງສານແບບກຸ່ມ, ການດໍາເນີນການຂອງໄອຍະການສູງສຸດແບບແບ່ງສ່ວນ, ການຮ້ອງຂໍຄໍາສັ່ງຫ້າມສາທາລະນະ ແລະ ການດໍາເນີນຄະດີອື່ນໆ ຫຼື ການຮ້ອງຂໍການບັນເທົາທຸກທີ່ຜູ້ໃດຜູ້ໜຶ່ງດໍາເນີນການໃນຄວາມສາມາດຂອງຕົວແທນແມ່ນບໍ່ໄດ້ຮັບອະນຸຍາດ. ຫຼື​ ບໍ່​ໃຫ້​ມີ​ການ​ລວມ​ການ​ຮ້ອງ​ຟ້ອງ​ບຸກ​ຄົນ ໂດຍ​ບໍ່​ມີ​ຄຳ​ຍິນຍອມ​ຂອງ​ທຸກ​ຝ່າຍ. ນອກຈາກນັ້ນ, ຖ້າຂໍ້ຂັດແຍ່ງບໍ່ແມ່ນສ່ວນໜຶ່ງຂອງຄະດີທີ່ກ່ຽວຂ້ອງທີ່ໄດ້ອະທິບາຍໄວ້ຂ້າງເທິງ, ຂໍ້ຂັດແຍ້ງມີໜ້ອຍກວ່າ $75,000 ແລະ ກ່ອນທີ່ຈະເລີ່ມການໄກ່ເກ່ຍ, ທ່ານປະຕິບັດຕາມເງື່ອນໄຂການໄກ່ເກ່ຍລ່ວງໜ້າທັງໝົດໃນພາກ 2.14 ນີ້, (1) ພວກເຮົາຈະຈ່າຍຄືນການຍື່ນຂອງທ່ານທັນທີໃນຄ່າທຳນຽມ ແລະ ຄ່າໃຊ້ຈ່າຍຂອງ AAA ແລະ ຜູ້ໄກ່ເກ່ຍ; ແລະ (2) ຖ້າການສະຫຼຸບຂອງສານໄກ່ເກ່ຍ, ຜູ້ໄກ່ເກ່ຍໃຫ້ລາງວັນແກທ່ານຫຼາຍກວ່າຂໍ້ສະເໜີທີ່ຂຽນຄັ້ງສຸດທ້າຍຂອງພວກເຮົາກ່ອນການແຕ່ງຕັ້ງຜູ້ໄກ່ເກ່ຍ, ພວກເຮົາຈະຈ່າຍເງິນໃຫ້ທ່ານ (i) ຈຳນວນລາງວັນ ຫຼື $1,000 USD (ອັນໃດກໍ່ຕາມທີ່ຫຼາຍກວ່ານັ້ນ); ແລະ (ii) ສໍາລັບຄ່າທໍານຽມທະນາຍຄວາມທີ່ສົມເຫດສົມຜົນທເກີດຂຶ້ນກັບທ່ານ; ແລະ ຄ່າໃຊ້ຈ່າຍທີ່ສົມເຫດສົມຜົນໃດໆ (ລວມທັງຄ່າທໍານຽມການເປັນພະຍານຜູ້ຊ່ຽວຊານ ແລະ ຄ່າໃຊ້ຈ່າຍຕ່າງໆ) ທີ່ເກີດຂຶ້ນກັບທະນາຍຄວາມຂອງທ່ານໃນສ່ວນທີ່ກ່ຽວຂ້ອງກັບການໄກ່ເກ່ຍສ່ວນບຸກຄົນຂອງທ່ານ.

(ສ)         ການ​ແຍກ​ອອກ​ມາ​ນຳ​ໃຊ້

. ຖ້າ, ຫຼັງຈາກໝົດການອຸທອນທັງໝົດ, ສານພົບວ່າພາກສ່ວນໃດໜຶ່ງຂອງພາກທີ 2.14 ນີ້ ບໍ່ສາມາດບັງຄັບໃຊ້ໄດ້ຕໍ່ກັບການຮ້ອງຟ້ອງ ຫຼື ການຮ້ອງຂໍການແກ້ໄຂໃດໆ, ຫຼັງຈາກນັ້ນ, ພາກສ່ວນທີ່ຕົກລົງເຫັນດີທີ່ຈະໄກ່ເກ່ຍການຮຽກຮ້ອງ ແລະ ການແກ້ໄຂທັງໝົດພາຍໃຕ້ການໄກ່ເກ່ຍ ກ່ອນທີ່ຈະດໍາເນີນຄະດີໃນສານການຮ້ອງຟ້ອງ ຫຼື ການແກ້ໄຂທີ່ຍັງເຫຼືອ (ເຊັ່ນ: ການຮ້ອງຂໍການແກ້ໄຂຄໍາສັ່ງສາທາລະນະ, ໃນກໍລະນີນີ້, ຜູ້ໄກ່ເກ່ຍອອກຄໍາຕັດສິນກ່ຽວກັບຄວາມຮັບຜິດຊອບ ແລະ ການຜ່ອນຜັນສ່ວນບຸກຄົນກ່ອນທີ່ສານຈະພິຈາລະນາຄໍາຮ້ອງຂໍນັ້ນ). ຖ້າບໍ່ດັ່ງນັ້ນ, ຖ້າພາກສ່ວນອື່ນໆຂອງພາກທີ 2.14 ພົບວ່າບໍ່ສາມາດບັງຄັບໄດ້, ສ່ວນທີ່ຍັງເຫຼືອຈະຍັງຄົງມີຜົນບັງຄັບໃຊ້ (ມີຄໍາຕັດສິນຊີ້ຂາດອອກໃຫ້ກ່ອນການດໍາເນີນຄະດີໃດໆຂອງສານເລີ່ມຕົ້ນ).

2.15.      ທັງ​ໝົດ​ຂໍ້​ຕົ​ກ​ລົງ

. ຂໍ້ຕົກລົງນີ້ (ຮ່ວມກັບເງື່ອນໄຂການອະນຸຍາດເຈ້ຍທີ່ພິມ ຫຼື ຂໍ້ກໍານົດອື່ນໆທີ່ມາພ້ອມກັບສ່ວນເສີມຊອບແວ, ການອັບເດດ ແລະ ການບໍລິການຕ່າງໆທີ່ພວກເຮົາສະໜອງໃຫ້ ແລະ ທີ່ທ່ານໃຊ້) ແລະ ຂໍ້ກໍານົດທີ່ມີຢູ່ໃນເວັບລິ້ງທີ່ລະບຸໄວ້ໃນຂໍ້ຕົກລົງນີ້ ແມ່ນຂໍ້ຕົກລົງທັງໝົດສໍາລັບ ຊອບແວ ແລະ ສ່ວນເສີມ, ການອັບເດດ ແລະ ການບໍລິການຕ່າງໆ (ເວັ້ນເສຍແຕ່ວ່າ ພວກເຮົາຈະໃຫ້ເງື່ອນໄຂອື່ນໆທີ່ມີສ່ວນເສີມ, ການອັບເດດ ຫຼື ການບໍລິການດັ່ງກ່າວ).

2.16.      ກົດໝາຍທີ່ນຳໃຊ້ ແລະ ສະຖານທີ່ເພື່ອແກ້ໄຂຂໍ້ຂັດແຍ່ງ

. ຖ້າທ່ານໄດ້ຊອຟແວມາໃນສະຫະລັດ ຫຼື ການາດາ, ກົດໝາຍຂອງລັດ ຫຼື ແຂວງທີ່ທ່ານອາໄສຢູ່ (ຫຼື ຖ້າທຸລະກິດ, ບ່ອນທີ່ສະຖານທີ່ທຸລະກິດຕົ້ນຕໍຂອງທ່ານຕັ້ງຢູ່) ປົກຄອງການຕີຄວາມໝາຍຂອງຂໍ້ຕົກລົງນີ້, ອ້າງສິດວ່າການລະເມີດຂອງມັນ ແລະ ການຮຽກຮ້ອງອື່ນໆທັງໝົດ (ລວມທັງການປົກປ້ອງຜູ້ບໍລິໂພກ, ການແຂ່ງຂັນທີ່ບໍ່ຍຸດຕິທໍາ ແລະ ການຮຽກຮ້ອງການລະເມີດ), ໂດຍບໍ່ຄໍານຶງເຖິງຂໍ້ຂັດແຍ່ງຂອງຫຼັກການຂອງກົດໝາຍ, ຍົກເວັ້ນວ່າ FAA ຄຸ້ມຄອງທຸກສິ່ງທຸກຢ່າງທີ່ກ່ຽວຂ້ອງກັບ​ການ​ໄກ່​ເກ່ຍ. ຖ້າ​ທ່ານ​ໄດ້​ຮັບຊອບແວນີ້​ມາ​ຢູ່​ໃນ​ປະ​ເທດ​ອື່ນ, ແມ່ນ​ນຳ​ໃຊ້​ກົດ​ໝາຍ​ຂອງ​ປະ​ເທດນັ້ນ, ຍົກ​ເວັ້ນ​ແຕ່ FAA ຄຸ້ມ​ຄອງ​ທຸກ​ຢ່າງ​ທີ່​ກ່ຽວ​ຂ້ອງ​ກັບ​ການ​ໄກ່​ເກ່ຍ. ຖ້າ​ມີ​ອຳ​ນາດ​ສານ​ລັດ​ຖະ​ບານ​ກາງ​ຂອງ​ສະ​ຫະ​ລັດ​ຢູ່, ທ່ານ ແລະ Microsoft ຍິນຍອມ​ຕໍ່​ອຳ​ນາດ​ສານ​ທັງ​ໝົດ ແລະ ສະ​ຖານ​ທີ່​ແມ່ນ​ຢູ່​ໃນ​ສານ​ລັດ​ຖະ​ບານ​ກາງ​ຢູ່​ໃນ King County, Washington ສໍາ​ລັບ​ຂໍ້​ຂັດ​ແຍ່ງ​ທັງ​ໝົດ​ທີ່​ບໍ່​ໄດ້​ຮັບ​ຟັງ​ໃນ​ການ​ຊີ້​ຂາດ ​ຫຼື​ ສານ​ຮ້ອງ​ຟ້ອງ​ຂະໜາດ​ນ້ອຍ. ຖ້າບໍ່ແມ່ນ, ທ່ານ ແລະ Microsoft ຍິນຍອມໃຫ້ສິດອຳນາດສະເພາະ ແລະ ສະຖານທີ່ຢູ່ໃນສານຊັ້ນສູງຂອງ King County, Washington ສໍາລັບຂໍ້ຂັດແຍ່ງທັງໝົດທີ່ບໍ່ໄດ້ຮັບການພິຈາລະນາໃນສານໄກ່ເກ່ຍ ຫຼື ສານ​ຮ້ອງ​ຟ້ອງ​ຂະໜາດ​ນ້ອຍ.

2.17.      ສິດ​ຂອງ​ຜູ້ບໍ​ລິ​ໂພກ; ການ​ປ່ຽນ​ແປງ​ໃນ​ພາກ​ພື້ນ

. ຂໍ້​ຕົກ​ລົງ​ນີ້​ອະ​ທິ​ບາຍ​ເຖິງ​ສິດ​ທາງ​ກົດ​ໝາຍ​ບາງປະການ. ທ່ານ​ອາດ​ຈະ​ມີ​ສິດ​ອື່ນ, ລວມ​ທັງ​ສິດ​ຜູ້​ບໍ​ລິ​ໂພກ​ພາຍ​ໃຕ້​ກົດ​ໝາຍ​ຂອງ​ລັດ, ແຂວງ ຫຼື ປະ​ເທດຂອງ​ທ່ານ. ທ່ານ​ຍັງ​ອາດ​ຈະ​ມີ​ສິດ​ກ່ຽວ​ກັບ​ຝ່າຍ​ຈາກ​ທີ່​ທ່ານ​ໄດ້​ຊອບ​ແວ​ມານັ້ນ​ຕື່ມ​ອີກ. ຂໍ້​ຕົກ​ລົງ​ນີ້​ບໍ່​ປ່ຽນ​ແປງ​ສິດ​ອື່ນ​ໆເຫຼົ່າ​ນັ້ນ ຖ້າ​ກົດ​ໝາຍ​ຂອງ​ລັດ, ແຂວງ ຫຼື ປະ​ເທດ​ຂອງ​ທ່ານ​ບໍ່​ອະ​ນຸ​ຍາດ. ຕົວ​ຢ່າງ: ຖ້າ​ທ່ານ​ໄດ້​ຮັບຊອບແວມາ​ຈາກ​ໜຶ່ງ​ໃນ​ພາກ​ພື້ນ​ດັ່ງ​ລຸ່ມ​ນີ້ ຫຼື ກົດ​ໝາຍ​ຂອງ​ປະ​ເທດ​ບັງ​ຄັບ​ນຳ​ໃຊ້, ບົດບັນ​ຍັດ​ຕໍ່​ໄປ​ນີ້​ແມ່ນມີຜົນ​ນຳ​ໃຊ້​ກັບ​ທ່ານ:

(ກ)         ອົດ​ສະ​ຕຣາ​ລີ

. ສໍາລັບຜູ້ບໍລິໂພກທີ່ອາໄສຢູ່ໃນອົດສະຕາລີ, ການບໍລິການຂອງພວກເຮົາມາພ້ອມກັບການຄໍ້າປະກັນທີ່ບໍ່ສາມາດຖືກຍົກເວັ້ນ ພາຍໃຕ້ກົດໝາຍຜູ້ບໍລິໂພກອົດສະຕຣາລີ. ສໍາລັບຄວາມລົ້ມເຫຼວທີ່ສໍາຄັນກັບການບໍລິການ, ທ່ານມີສິດ:

(i)         ເພື່ອຍົກເລີກສັນຍາການບໍລິການຂອງທ່ານກັບພວກເຮົາ; ແລະ

(ii)         ການຄືນເງິນສໍາລັບສ່ວນທີ່ບໍ່ໄດ້ໃຊ້ ຫຼື ການຊົດເຊີຍສໍາລັບມູນຄ່າທີ່ຫຼຸດລົງຂອງມັນ.

2.18.      ການປະຕິເສດການຮັບປະກັນ

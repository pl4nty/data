หลังจากติดตั้งแล้ว คุณสามารถดูเงื่อนไขเหล่านี้ได้ใน Microsoft Edge ที่ edge://terms

หากคุณอยู่ใน (หรือเป็นธุรกิจที่มีสำนักงานใหญ่อยู่ใน) สหรัฐอเมริกา โปรดอ่านเนื้อหาในส่วน “การผูกพันตามคำชี้ขาดของอนุญาโตตุลาการ และการสละสิทธิ์ดำเนินคดีแบบกลุ่ม” ในส่วนที่ 2.14 ด้านล่างนี้ มีผลกระทบต่อวิธีการแก้ไขข้อพิพาท

นี่คือข้อตกลงใบอนุญาตระหว่างคุณและ Microsoft Corporation (หรือบริษัทในเครือ) ซึ่งอธิบายถึงสิทธิ ภาระผูกพัน และเงื่อนไขในการใช้ซอฟต์แวร์และการอัปเดตใด ๆ ("ซอฟต์แวร์") ของคุณ (ยกเว้นในกรณีที่การอัปเดตมาพร้อมกับข้อตกลงใหม่ หรือข้อตกลงเพิ่มเติม ซึ่งในกรณีนี้ เงื่อนไขที่แตกต่างเหล่านั้นจะใช้ได้ในอนาคตและจะไม่เปลี่ยนแปลงสิทธิของคุณหรือของ Microsoft ที่เกี่ยวข้องกับซอฟต์แวร์ที่อัปเดตไว้ล่วงหน้า ) คุณควรตรวจทานข้อตกลงทั้งหมด รวมถึงข้อกำหนดใดๆ ที่เกี่ยวข้อง เนื่องจากข้อกำหนดทั้งหมดนี้มีความสำคัญและก่อให้เกิดข้อตกลงฉบับนี้ที่ใช้บังคับกับคุณ

การยอมรับข้อตกลงนี้ หรือการดาวน์โหลด หรือการใช้ซอฟต์แวร์นี้เท่ากับว่าคุณยอมรับข้อกำหนดเหล่านี้ทุกประการ และยินยอมให้มีการส่งข้อมูลบางอย่างในระหว่างการใช้ซอฟต์แวร์ของคุณตามคำชี้แจงสิทธิ์ส่วนบุคคลซึ่งอธิบายไว้ในเงื่อนไขการอนุญาตให้ใช้สิทธิของ Windows ของคุณหรือในส่วนที่ 2.7 ด้านล่าง ถ้าหากว่าคุณไม่ยอมรับและปฏิบัติตามเงื่อนไขเหล่านี้ คุณจะไม่สามารถใช้ซอฟต์แวร์หรือคุณสมบัติของซอฟต์แวร์ได้

ข้อตกลงฉบับนี้ประกอบรวมด้วยสองส่วน: ส่วนที่ 1 มีผลใช้บังคับหากคุณใช้ซอฟต์แวร์บนแพลตฟอร์ม Windows ใดก็ตาม ส่วนที่ 2 ใช้ได้หากคุณใช้บนแพลตฟอร์มที่ไม่ใช่ Windows รวมถึง MacOS, iOS, Android และ Linux

1.     MICROSOFT EDGE สำหรับอุปกรณ์ WINDOWS

1.1.        เงื่อนไขการอนุญาตให้ใช้สิทธิสำหรับ Windows มีผลใช้บังคับ

. เงื่อนไขการอนุญาตให้ใช้สิทธิสำหรับซอฟต์แวร์ของ Microsoft, ระบบปฏิบัติการ Windows (“เงื่อนไขการอนุญาตให้ใช้สิทธิสำหรับ Windows”) ของคุณกำกับดูแลการใช้งานซอฟต์แวร์นี้ของคุณบนแพลตฟอร์ม Windows และเป็นส่วนหนึ่งของข้อตกลงฉบับนี้ คุณสามารถดูเงื่อนไขการอนุญาตให้ใช้สิทธิสำหรับ Windows ได้โดยไปที่ https://aka.ms/useterms หรือไปที่ การตั้งค่า (Settings) - ระบบ (System) - เกี่ยวกับ (About) ภายใน Windows

1.1.1.        การอัปเดต

. ซอฟต์แวร์จะตรวจหาโปรแกรมปรับปรุงสำหรับซอฟต์แวร์เป็นระยะ ๆ และจะดาวน์โหลดและติดตั้งโปรแกรมปรับปรุงเหล่านี้ให้กับคุณ คุณสามารถขอรับโปรแกรมปรับปรุงได้จาก Microsoft หรือแหล่งที่มาที่ได้รับอนุญาตเท่านั้น และ Microsoft อาจจำเป็นต้องปรับปรุง ระบบของคุณเพื่อให้คุณได้รับโปรแกรมปรับปรุงดังกล่าว การยอมรับข้อตกลงนี้หรือการใช้ซอฟต์แวร์ เท่ากับว่าคุณตกลงที่จะรับการปรับปรุงอัตโนมัติในลักษณะเหล่านี้โดยไม่มีการบอกกล่าวใด ๆ เพิ่มเติม

1.2.        ตัวอย่างแนะนำ

. คุณสามารถเลือกใช้ตัวอย่างแนะนำ รุ่นสำหรับคนภายใน รุ่นเบต้าหรือรุ่นก่อนวางจำหน่าย (“ตัวอย่างแนะนำ”) ที่ Microsoft จัดหาไว้ให้สำหรับคุณ คุณสามารถใช้ตัวอย่างแนะนำได้เฉพาะจนถึงวันหมดอายุของซอฟต์แวร์ และตราบเท่าที่คุณปฏิบัติตามข้อกำหนดการอนุญาตให้ใช้สิทธิ์เหล่านี้ ตัวอย่างแนะนำจะเป็นผลิตภัณฑ์ทดลองและอาจจะแตกต่างอย่างมากจากผลิตภัณฑ์ที่วางจำหน่าย แม้ว่าจะมีข้อขัดแย้งในเงื่อนไขการอนุญาตให้ใช้สิทธิสำหรับ Windows ตัวอย่างแนะนำจะได้รับการจัดหาให้ “ตามลักษณะที่เป็น” และไม่มีการรับประกันทั้งโดยนัยหรือโดยแจ้ง (รวมถึงการรับประกันที่มีข้อจำกัด) กับรุ่นเหล่านี้ การติดตั้งตัวอย่างซอฟต์แวร์บนอุปกรณ์ของคุณอาจทำให้การรับประกันอุปกรณ์ของคุณเป็นโมฆะ หรือทำให้มีผลกระทบต่อการรับประกันอุปกรณ์ของคุณ และอาจไม่มีสิทธิ์ได้รับการสนับสนุนจากผู้ผลิตอุปกรณ์ของคุณหรือผู้ให้บริการเครือข่าย (ถ้ามี) Microsoft จะไม่รับผิดชอบต่อความเสียหายใด ๆ อันเกิดจากคุณ Microsoft อาจจะไม่ให้บริการสนับสนุนสำหรับตัวอย่างแนะนำ ถ้าหากว่าคุณแสดงความคิดเห็น ข้อแนะนำ หรือคำติชมอื่น ๆ ให้ Microsoft เกี่ยวกับตัวอย่างแนะนำ (“การส่งความคิดเห็น”) จะถือว่าคุณมอบสิทธิ์ให้แก่ Microsoft และพันธมิตรของ Microsoft ในการใช้การส่งความคิดเห็นในลักษณะใดก็ตามและเพื่อวัตถุประสงค์ใดก็ตาม

1.2.1.        การเก็บรวบรวมข้อมูลสำหรับตัวอย่างแนะนำ

. สำหรับตัวอย่างแนะนำ การตั้งค่าความเป็นส่วนตัวและคุณสมบัติอาจจะไม่ทำงานตามที่ต้องการ และตัวอย่างแนะนำอาจไม่ทำงานกับการตั้งค่าความเป็นส่วนตัวอื่น ๆ ของ Windows รวมถึงการตั้งค่าข้อมูลการวิเคราะห์สำหรับ Windows 10 หรือ Windows 11 ข้อมูลที่เก็บรวบรวมจากการที่คุณใช้ตัวอย่างแนะนำ รวมถึงรายงานการวิเคราะห์ เทคนิค ข้อผิดพลาด การหยุดทำงาน และข้อมูลอื่น ๆ ที่เกี่ยวข้องจากอุปกรณ์ของคุณที่เปิดใช้ตัวอย่างแนะนำ อาจจะถูกใช้งาน จัดเก็บ ประมวล และวิเคราะห์ เพื่อทำให้ Windows, ซอฟต์แวร์นี้ และตัวอย่างแนะนำมีข้อมูลที่เป็นปัจจุบัน ปลอดภัย และทำงานได้อย่างเหมาะสม การเก็บข้อมูลยังช่วยให้เราสามารถพัฒนาปรับปรุงผลิตภัณฑ์และบริการของ Microsoft และอาจถูกใช้เพื่อวัตถุประสงค์อื่นใดที่อธิบายไว้ในคำแถลงเกี่ยวกับความเป็นส่วนตัวของ Microsoft หากคุณปิดใช้งานการเก็บรวบรวมข้อมูลผ่านการควบคุมที่มีในตัวอย่างแนะนำ เราอาจจะยังเก็บรวบรวมข้อมูลการวิเคราะห์เกี่ยวกับการดาวน์โหลด การติดตั้ง และการถอนตัวอย่างแนะนำ รวมถึงข้อมูล “พื้นฐาน” ดังที่อธิบายไว้ในหัวข้อการวิเคราะห์ของ Windows ในคำแถลงเกี่ยวกับความเป็นส่วนตัวของ Microsoft ที่ https://aka.ms/privacy

1.3.        บริการอื่น ๆ

. ซอฟต์แวร์อาจประกอบด้วยแอปหรือคุณสมบัติที่มีจุดเข้าถึงบริการออนไลน์ หรืออาศัยบริการออนไลน์ และการใช้บริการเหล่านั้นในบางครั้งจะอยู่ภายใต้ข้อกำหนดและนโยบายความเป็นส่วนตัวที่แยกต่างหาก เช่น ข้อตกลงการใช้บริการของ Microsoft ที่ https://aka.ms/msa คุณสามารถดูเงื่อนไขและนโยบายเหล่านี้ได้โดยดูที่เงื่อนไขการใช้บริการหรือการตั้งค่าของโปรแกรม แล้วแต่กรณี บริการอาจไม่ได้จัดไว้ให้ในทุกภูมิภาค คุณจะต้องไม่ใช่โทเค็นของซอฟต์แวร์เพื่อโทรเข้าไปที่บริการ Microsoft Azure ที่แยกออกจากซอฟต์แวร์

2.     MICROSOFT EDGE สำหรับอุปกรณ์ที่ไม่ใช่ของ WINDOWS

2.1.        สิทธิในการติดตั้งและใช้งาน

. ซอฟต์แวร์นี้เป็นการอนุญาตให้ใช้สิทธิ ไม่ใช่การขาย ภายใต้ข้อตกลงนี้ เราจะให้สิทธิ์แก่คุณในการติดตั้งและใช้งานอินสแตนซ์ของซอฟต์แวร์ได้หนึ่งอินสแตนซ์บนอุปกรณ์ของคุณสำหรับการใช้โดยบุคคลคนเดียวในหนึ่งครั้ง โดยใช้แพลตฟอร์มที่ไม่ใช่ Windows รวมถึง MacOS, iOS, Android และ Linux ตราบเท่าที่คุณปฏิบัติตามเงื่อนไขและข้อจำกัดทั้งหมดของข้อตกลงนี้เท่านั้น

2.2.        ซอฟต์แวร์ของบุคคลภายนอก

. ซอฟต์แวร์นี้อาจประกอบด้วยโปรแกรมของบุคคลภายนอกที่ให้สิทธิ์การใช้งานแก่คุณภายใต้ข้อตกลงนี้ หรือภายใต้ข้อกำหนดของตนเอง ถ้าหากว่ามีข้อกำหนดการอนุญาตให้ใช้สิทธิ์ การบอกกล่าวและการรับรู้สำหรับโปรแกรมของบุคคลภายนอก จะสามารถดูได้ที่ https://aka.ms/thirdpartynotices ถึงแม้โปรแกรมดังกล่าวจะอยู่ภายใต้ข้อตกลงอื่นก็ตาม การปฏิเสธ การจำกัด และการยกเว้นความรับผิดต่อความเสียหายด้านล่างนี้ก็ยังนำไปใช้ได้ภายในขอบเขตที่กฎหมายอนุญาต

2.3.        ตัวอย่างแนะนำ

. คุณสามารถเลือกใช้ตัวอย่างแนะนำ รุ่นสำหรับคนภายใน รุ่นเบต้าหรือรุ่นก่อนวางจำหน่าย (“ตัวอย่างแนะนำ”) ที่ Microsoft จัดหาไว้ให้สำหรับคุณ คุณสามารถใช้ตัวอย่างแนะนำได้เฉพาะจนถึงวันหมดอายุของซอฟต์แวร์ และตราบเท่าที่คุณปฏิบัติตามข้อกำหนดการอนุญาตให้ใช้สิทธิ์เหล่านี้ ตัวอย่างแนะนำจะเป็นผลิตภัณฑ์ทดลองและอาจจะแตกต่างอย่างมากจากผลิตภัณฑ์ที่วางจำหน่าย ตัวอย่างแนะนำจะได้รับการจัดหาให้ “ตามลักษณะที่เป็น” และไม่มีการรับประกันทั้งโดยนัยหรือโดยแจ้งกับรุ่นเหล่านี้ การติดตั้งตัวอย่างซอฟต์แวร์บนอุปกรณ์ของคุณอาจทำให้การรับประกันอุปกรณ์ของคุณเป็นโมฆะ หรือทำให้มีผลกระทบต่อการรับประกันอุปกรณ์ของคุณ และอาจไม่มีสิทธิ์ได้รับการสนับสนุนจากผู้ผลิตอุปกรณ์ของคุณหรือผู้ให้บริการเครือข่าย (ถ้ามี) Microsoft จะไม่รับผิดชอบต่อความเสียหายใด ๆ อันเกิดจากคุณ Microsoft อาจจะไม่ให้บริการสนับสนุนสำหรับตัวอย่างแนะนำ ถ้าหากว่าคุณแสดงความคิดเห็น ข้อแนะนำ หรือคำติชมอื่น ๆ ให้ Microsoft เกี่ยวกับตัวอย่างแนะนำ (“การส่งความคิดเห็น”) จะถือว่าคุณมอบสิทธิ์ให้แก่ Microsoft และพันธมิตรของ Microsoft ในการใช้การส่งความคิดเห็นในลักษณะใดก็ตามและเพื่อวัตถุประสงค์ใดก็ตาม

. สำหรับตัวอย่างแนะนำที่อยู่ภายใต้ส่วนที่ 2.3 การตั้งค่าความเป็นส่วนตัวและคุณสมบัติอาจจะไม่ทำงานตามที่ต้องการ และตัวอย่างแนะนำอาจไม่ทำงานกับการตั้งค่าความเป็นส่วนตัวของคุณของระบบปฏิบัติการอื่น ๆ ข้อมูลที่เก็บรวบรวมจากการที่คุณใช้ตัวอย่างแนะนำ รวมถึงรายงานการวิเคราะห์ เทคนิค ข้อผิดพลาด การหยุดทำงาน และข้อมูลอื่น ๆ ที่เกี่ยวข้องจากอุปกรณ์ของคุณที่เปิดใช้ตัวอย่างแนะนำ อาจจะถูกใช้งาน จัดเก็บ ประมวล และวิเคราะห์ เพื่อทำให้ตัวอย่างแนะนำมีข้อมูลที่เป็นปัจจุบัน ปลอดภัย และทำงานได้อย่างเหมาะสม การเก็บข้อมูลยังช่วยให้เราสามารถพัฒนาปรับปรุงผลิตภัณฑ์และบริการของ Microsoft และอาจถูกใช้เพื่อวัตถุประสงค์อื่นใดที่อธิบายไว้ในคำแถลงเกี่ยวกับความเป็นส่วนตัวของ Microsoft หากคุณปิดใช้งานการเก็บรวบรวมข้อมูลผ่านการควบคุมที่มีในตัวอย่างแนะนำ เราอาจจะยังเก็บรวบรวมข้อมูลการวิเคราะห์เกี่ยวกับการดาวน์โหลด การติดตั้ง และการถอนตัวอย่างแนะนำ รวมถึงข้อมูล “พื้นฐาน” ดังที่อธิบายไว้ในหัวข้อการวิเคราะห์ของ Windows ในคำแถลงเกี่ยวกับความเป็นส่วนตัวของ Microsoft https://aka.ms/privacy

2.4.        บริการอื่น ๆ

. ซอฟต์แวร์อาจประกอบด้วยคุณสมบัติหรือแอปที่มีจุดเข้าถึงบริการออนไลน์ หรืออาศัยบริการออนไลน์ และการใช้บริการเหล่านั้นในบางครั้งจะอยู่ภายใต้ข้อกำหนดและนโยบายความเป็นส่วนตัวที่แยกต่างหาก เช่น ข้อตกลงการใช้บริการของ Microsoft ที่ https://aka.ms/msa คุณสามารถดูเงื่อนไขและนโยบายเหล่านี้ได้โดยดูที่เงื่อนไขการใช้บริการหรือการตั้งค่าของโปรแกรม แล้วแต่กรณี บริการอาจไม่ได้จัดไว้ให้ในทุกภูมิภาค

2.5.        การสื่อสารกับคุณ

. Microsoft อาจใช้ข้อมูลติดต่อของคุณ (i) เพื่อสื่อสารกับคุณเกี่ยวกับการใช้ซอฟต์แวร์ของคุณ และ (ii) เพื่อมอบข้อมูลเพิ่มเติมให้แก่คุณ เกี่ยวกับซอฟต์แวร์และผลิตภัณฑ์หรือบริการอื่น ๆ ของ Microsoft การติดต่อนี้อาจจะเป็นทางอีเมล SMS ข้อความโต้ตอบ การแชตบทเว็บ โทรศัพท์ ในอินเตอร์เฟซของผู้ใช้ หรือวิธีการอื่น ๆ และอาจรวมถึงข้อเสนอต่าง ๆ คุณเลือกได้เสมอว่าต้องการรับอีเมลส่งเสริมการขาย ข้อความ SMS การโทรศัพท์ และไปรษณีย์จาก Microsoft หรือไม่

2.6.        ผลตอบรับ

. หากคุณส่งข้อคิดเห็น ข้อแนะนำ หรือคำติชมอื่น ๆ ให้ Microsoft เกี่ยวกับซอฟต์แวร์ (“การส่งความคิดเห็น”) ถึง Microsoft จะถือว่าคุณมอบสิทธิให้แก่ Microsoft และพันธมิตรของ Microsoft ในการใช้การส่งความคิดเห็นในลักษณะใดก็ตามและเพื่อวัตถุประสงค์ใดก็ตาม คุณจะไม่ส่งความคิดเห็นที่เกี่ยวข้องกับการอนุญาตให้ใช้สิทธิที่กำหนดให้ Microsoft ต้องให้สิทธิการใช้งานในซอฟต์แวร์หรือเอกสารประกอบของตนแก่บุคคลภายนอก เนื่องจาก Microsoft ได้รวมความเห็นของคุณไว้ในเอกสารดังกล่าว สิทธิเหล่านี้จะมีผลต่อไปแม้ข้อตกลงนี้สิ้นสุดลงแล้ว

2.7.         สิทธิความเป็นส่วนตัว ความยินยอมให้ใช้ข้อมูล

. สิทธิความเป็นส่วนตัวของคุณมีความสำคัญกับเรา คุณสมบัติของซอฟต์แวร์บางอย่างจะส่งหรือรับข้อมูลเมื่อใช้คุณสมบัติเหล่านั้น โดยการยอมรับข้อตกลงนี้และการใช้ซอฟต์แวร์นี้ เท่ากับคุณตกลงว่า Microsoft สามารถเก็บรวบรวม ใช้ และเปิดเผยข้อมูลได้ตามที่อธิบายไว้ใน คำชี้แจงสิทธิ์ส่วนบุคคลของ Microsoft ที่ https://aka.ms/privacy และตามที่อาจได้อธิบายไว้ในส่วนติดต่อผู้ใช้ที่มาพร้อมกับคุณสมบัติของซอฟต์แวร์

2.8.         ข้อจำกัด

. Microsoft จึงสงวนสิทธิ์ทุกประการ (เช่น สิทธิ์ภายใต้กฎหมายทรัพย์สินทางปัญญา) ที่มิได้มีการมอบให้อย่างชัดแจ้งในข้อตกลงนี้ และไม่มีการอนุญาตให้คุณใช้สิทธิอื่นใด ทั้งนี้เพื่อหลีกเลี่ยงความสับสน สิทธิ์ใช้งานนี้ไม่ได้เป็นการมอบสิทธิ์ใด ๆ ให้แก่คุณ และคุณจะต้องไม่ (และไม่อนุญาตให้บุคคลหรือนิติบุคคลอื่นใด):

(a)    ใช้หรือทำให้คุณสมบัติซอฟต์แวร์มีลักษณะเป็นแบบเสมือนแยกออกมา;

(b)    เผยแพร่ ทำสำเนา (ยกเว้นการทำสำเนาสำรองเพียงชุดเดียว) ให้เช่า ให้เช่าซื้อ หรือให้ยืมซอฟต์แวร์;

(c)    เข้าไปยุ่งเกี่ยวกับข้อจำกัดทางด้านเทคนิคใด ๆ ในซอฟต์แวร์ที่อนุญาตให้คุณใช้ในรูปแบบเฉพาะ;

(d)    ทำวิศวกรรมย้อนกลับ แยกส่วน หรือแยกองค์ประกอบซอฟต์แวร์ หรือพยายามทำเช่นนั้น ยกเว้นและเฉพาะภายใต้เงื่อนไขที่ว่าข้อจำกัดที่กล่าวมาก่อนหน้านี้ (i) ได้รับการอนุญาตตามกฎหมายที่ใช้บังคับ (ii) ได้รับการอนุญาตตามเงื่อนไขการอนุญาตให้ใช้สิทธิ์ที่ว่าด้วยการใช้คอมโพเนนต์โอเพนซอร์ส (Open Source) ซึ่งอาจรวมอยู่ในซอฟต์แวร์ หรือ (iii) มีความจำเป็นเพื่อแก้จุดบกพร่องในการเปลี่ยนแปลงที่ดำเนินการกับไลบรารีใด ๆ ที่ได้รับสิทธิการใช้งานภายใต้ GNU Lesser General Public License ซึ่งอาจรวมอยู่กับซอฟต์แวร์และมีการเชื่อมโยงโดยซอฟต์แวร์

(e)    เมื่อใช้คุณสมบัติทางอินเทอร์เน็ต คุณจะต้องไม่ใช้คุณสมบัติเหล่านั้นไม่ว่าในรูปแบบใดก็ตามที่อาจไปรบกวนการใช้งานคุณสมบัติเหล่านั้นของผู้อื่น หรือพยายามเข้าถึงหรือใช้บริการ ข้อมูล บัญชี หรือเครือข่ายใดก็ตามโดยที่ไม่ได้รับอนุญาต;

(f)     ใช้โทเค็นของซอฟต์แวร์เพื่อโทรเข้าไปที่บริการ Microsoft Azure ที่แยกออกจากซอฟต์แวร์

2.9.         เอกสารประกอบ

. ในกรณีที่มีเอกสารกำกับมาพร้อมกับซอฟต์แวร์นั้น คุณสามารถทำสำเนาและใช้เอกสารกำกับเพื่อการอ้างอิงเป็นการภายในได้

2.10.      มาตรฐานภาพ H.264/AVC และ MPEG-4 และมาตรฐานวิดีโอ VC-1

. ซอฟต์แวร์อาจมีเทคโนโลยี codec ภาพ H.264/AVC, VC-1 และ MPEG-4 MPEG LA, L.L.C. กำหนดให้ต้องใช้คำประกาศนี้:
ผลิตภัณฑ์นี้ได้รับอนุญาตตามใบอนุญาตสิทธิบัตร AVC, VC-1 และภาพ MPEG-4 PART 2 สำหรับใช้งานส่วนบุคคลที่ไม่ใช่เชิงพาณิชย์ของผู้บริโภคเพื่อการ (i) เข้ารหัสวิดีโอตามมาตรฐานข้างต้น (“มาตรฐานวิดีโอ”) และ/หรือ (ii) ถอดรหัสวิดีโอ AVC, VC-1 และ MPEG-4 PART 2 ที่ถูกเข้ารหัสโดยผู้บริโภคเพื่อวัตถุประสงค์ส่วนบุคคลที่ไม่ใช่เชิงพาณิชย์ หรือที่ได้รับจากผู้จัดหาวิดีโอที่ได้รับอนุญาตให้จัดหาวิดีโอดังกล่าว ทั้งนี้ ไม่มีการให้อนุญาตหรือบอกเป็นนัยถึงการให้อนุญาตสำหรับการใช้งานอื่น ๆ คุณสามารถดูข้อมูลเพิ่มเติมได้จาก MPEG LA, L.L.C โปรดดูที่ https://aka.ms/mpegla

2.11.      ข้อจำกัดในการส่งออก

. คุณต้องปฏิบัติตามกฎหมายและข้อบังคับด้านการส่งออกทั้งในและระหว่างประเทศที่ใช้บังคับกับซอฟต์แวร์ ซึ่งรวมถึงข้อจำกัดของที่หมายปลายทาง ผู้ใช้ปลายทาง และการใช้ขั้นปลาย สำหรับข้อมูลเพิ่มเติมเกี่ยวกับข้อจำกัดในการส่งออก โปรดดูที่ https://aka.ms/exporting

2.12.      บริการสนับสนุน

. Microsoft ไม่มีข้อผูกพันภายใต้ข้อตกลงฉบับนี้ที่จะให้บริการสนับสนุนสำหรับซอฟต์แวร์นี้ การสนับสนุนใด ๆ ที่ให้นั้นเป็น “อย่างที่เป็น” “มีข้อผิดพลาดทั้งหมด” และไม่มีการรับประกันใด ๆ ทั้งสิ้น

2.13.      การอัปเดต

. ซอฟต์แวร์จะตรวจหาโปรแกรมปรับปรุงสำหรับซอฟต์แวร์เป็นระยะ ๆ และจะดาวน์โหลดและติดตั้งโปรแกรมปรับปรุงเหล่านี้ให้กับคุณ คุณสามารถขอรับโปรแกรมปรับปรุงได้จาก Microsoft หรือแหล่งที่มาที่ได้รับอนุญาตเท่านั้น และ Microsoft อาจจำเป็นต้องปรับปรุงระบบของคุณเพื่อให้คุณได้รับโปรแกรมปรับปรุงดังกล่าว การยอมรับข้อตกลงนี้หรือการใช้ซอฟต์แวร์ เท่ากับว่าคุณตกลงที่จะรับการปรับปรุงอัตโนมัติในลักษณะเหล่านี้โดยไม่มีการบอกกล่าวใด ๆ เพิ่มเติม

2.14.      การวินิจฉัยโดยอนุญาโตตุลาการที่มีผลผูกพันและการสละสิทธิ์การดำเนินคดีแบบกลุ่ม

. เราหวังว่าเราจะไม่มีพิพาทเกิดขึ้น แต่หากว่าเรามี คุณและเราตกลงกันที่จะพยายามระงับข้อพิพาทในเวลา 60 วัน เมื่อได้รับหนังสือแจ้งข้อพิพาท เพื่อแก้ไขอย่างไม่เป็นทางการ ถ้าหากว่าเราไม่สามารถระงับข้อพิพาทนั้นได้ เราตกลงว่าจะ
ใช้การวินิจฉัยโดยอนุญาโตตุลาการที่มีผลผูกพันเป็นรายบุคคลต่อหน้าสมาคมอนุญาโตตุลาการแห่งอเมริกา ("AAA") ภายใต้พระราชบัญญัติอนุญาโตตุลาการแห่งสหพันธรัฐ ("FAA") และไม่ฟ้องต่อศาลต่อหน้าผู้พิพากษาหรือคณะลูกขุน แต่อนุญาโตตุลาการที่เป็นกลางจะเป็นผู้ตัดสินและคำตัดสินของอนุญาโตตุลาการจะถือเป็นที่สุด ยกเว้นสิทธิที่จำกัดในการอุทธรณ์ภายใต้ FAA ทั้งนี้ ไม่อนุญาตให้มีการฟ้องร้องดำเนินคดีแบบกลุ่ม การอนุญาโตตุลาการทั้งชั้นเรียน การดำเนินการของอัยการสูงสุดเอกชน การร้องขอคำสั่งห้ามสาธารณะ และการดำเนินการหรือการร้องขอการเยียวยาอื่นใดที่มีผู้กระทำการในฐานะตัวแทน การนำกระบวนพิจารณาที่เป็นเอกเทศมารวมกันโดยไม่ได้รับความยินยอมจากคู่ความทุกฝ่ายก็ไม่สามารถทำได้เช่นกัน “เรา” และ “ของเรา” จะรวมถึง Microsoft และบริษัทในเครือของ Microsoft

(a)         ครอบคลุมข้อพิพาททุกประเด็น ยกเว้นทรัพย์สินทางปัญญา (IP)

. คำว่า “ข้อพิพาท” จะมีความหมายกว้างมากที่สุดเท่าที่จะสามารถเป็นไปได้ ซึ่งจะรวมถึงการเรียกร้องสิทธิ์หรือข้อโต้เถียงใด ๆ ระหว่างคุณกับเราในเรื่องที่เกี่ยวข้องกับซอฟต์แวร์ ราคาของซอฟต์แวร์ การตลาด การสื่อสาร ธุรกรรมการซื้อของคุณ การเรียกเก็บเงิน หรือข้อตกลงนี้ ภายใต้ทฤษฎีทางกฎหมายใด ๆ รวมถึงสัญญา การรับประกัน การละเมิด บทกฎหมาย หรือข้อบังคับ ยกเว้นข้อพิพาทที่เกี่ยวข้องกับการบังคับใช้หรือความถูกต้องของคุณ ผู้อนุญาตของคุณ สิทธิ์ในทรัพย์สินทางปัญญาของเรา หรือของผู้อนุญาต

(b)         ส่งหนังสือแจ้งข้อพิพาทก่อนอนุญาโตตุลาการ

. หากว่าคุณมีข้อพิพาทที่ตัวแทนฝ่ายบริการลูกค้าของเราไม่สามารถแก้ไขได้ และคุณต้องการดำเนินการอนุญาโตตุลาการ คุณจะต้องส่งหนังสือแจ้งข้อพิพาทเป็นรายบุคคลโดย Microsoft Corporation, ATTN: CELA ARBITRATION, One Microsoft Way, Redmond, WA 98052-6399 หรือส่งแบบฟอร์มทางอิเล็กทรอนิกส์ แบบฟอร์มการแจ้งข้อพิพาทมีอยู่ที่ https://go.microsoft.com/fwlink/?LinkId=245499 กรอกแบบฟอร์มนั้นให้ครบถ้วน พร้อมข้อมูลทั้งหมดที่จำเป็น เราจะทำแบบเดียวกันในกรณีที่เรามีข้อพิพาทกับคุณ หนังสือแจ้งข้อพิพาทนี้เป็นข้อกำหนดเบื้องต้นในการเริ่มอนุญาโตตุลาการใด ๆ อายุความที่เกี่ยวข้องใด ๆ จะถูกเรียกเก็บนับจากวันที่ยื่นหนังสือแจ้งข้อพิพาทรายบุคคลอย่างถูกต้อง จนถึงวันแรกที่สามารถยื่นคำตัดสินโดยอนุญาโตตุลาการได้อย่างถูกต้องภายใต้ส่วนที่ 2.14 นี้

(c)         ตัวเลือกที่เป็นศาลเรียกค่าเสียหายเล็กน้อย (Small claims court)

. แทนที่จะส่งหนังสือแจ้งข้อพิพาท คุณหรือเราอาจฟ้องอีกฝ่ายในศาลคดีมโนสาเร่ที่ต้องการการบรรเทาทุกข์เป็นรายบุคคลเท่านั้น ตราบใดที่การดำเนินการดังกล่าวเป็นไปตามข้อกำหนดของศาลคดีมโนสาเร่ และยังคงเป็นการดำเนินการของบุคคลที่ต้องการการบรรเทาทุกข์เป็นรายบุคคล ศาลคดีมโนสาเร่จะต้องอยู่ในเขตที่คุณอาศัยอยู่ (หรือสถานประกอบธุรกิจหลักของคุณ ในกรณีที่เป็นองค์กรธุรกิจ)

(d)         กระบวนการอนุญาโตตุลาการ

. AAA จะดำเนินกระบวนการอนุญาโตตุลาการใด ๆ ภายใต้ข้อบังคับอนุญาโตตุลาการการค้า (Commercial Arbitration Rules) ของ AAA (หรือในกรณีที่คุณเป็นบุคคลและใช้ซอฟต์แวร์สำหรับการใช้งานส่วนบุคคลหรือการใช้งานในครัวเรือน หรือในกรณีที่จำนวนทุนทรัพย์ของข้อพิพาทไม่เกิน 75,000 ดอลลาร์สหรัฐฯ ไม่ว่าคุณจะเป็นบุคคลหรือไม่หรือใช้ซอฟต์แวร์อย่างไร ข้อบังคับอนุญาโตตุลาการสำหรับผู้บริโภค (Consumer Arbitration Rules) ของ AAA ก็จะมีผลใช้บังคับด้วย) กฎเพิ่มเติมสำหรับการอนุญาโตตุลาการโดยรวมของ AAA ซึ่งได้รับการแก้ไขโดยส่วนที่ 2.14 นี้จะนำไปใช้ในกรณีที่เกี่ยวข้อง (ตามที่กล่าวไว้ด้านล่าง) สำหรับข้อมูลเพิ่มเติม โปรดดูที่ https://aka.ms/adr ข้อตกลงนี้ใช้บังคับในขอบเขตที่ขัดแย้งกับกฎ AAA ที่เกี่ยวข้อง หากว่าต้องการเริ่มการอนุญาโตตุลาการ ให้ยื่นแบบฟอร์มการเรียกร้องอนุญาโตตุลาการซึ่งมีอยู่ที่ https://go.microsoft.com/fwlink/?LinkId=245497 ไปยัง AAA และส่งสำเนาทางไปรษณีย์มายังเรา แบบฟอร์มจะต้องมีข้อมูลเฉพาะสำหรับคุณแลการอ้างสิทธิ์ของคุณ สำหรับข้อพิพาทที่มีจำนวนทุนทรัพย์ไม่เกิน 25,000 ดอลลาร์สหรัฐฯ การพิจารณาคดีจะดำเนินการผ่านทางโทรศัพท์หรือการประชุมทางวิดีโอ เว้นแต่อนุญาโตตุลาการจะพบเหตุอันสมควรที่จะจัดให้มีการพิจารณาคดีต่อหน้าคู่ความแทน การพิจารณาคดีต่อหน้าบุคคลจะเกิดขึ้นในเขตพื้นที่ที่คุณอาศัยอยู่ (หรือสถานประกอบธุรกิจหลักของคุณ ในกรณีที่เป็นองค์กรธุรกิจ) อนุญาโตตุลาการอาจมีคำตัดสินชี้ขาดให้ชดใช้ค่าเสียหายแก่คุณเป็นรายบุคคลในจำนวนเดียวกับที่ศาลอาจตัดสิน อนุญาโตตุลาการอาจมีคำสั่งแสดงสิทธิหรือคำสั่งคุ้มครองชั่วคราวให้กับคุณเป็นรายบุคคลเพื่อให้บรรลุตามการเรียกร้องสิทธิ์ของคุณเท่านั้น แต่ไม่ใช่การบรรเทาทุกข์ที่จะส่งผลกระทบต่อบุคคลอื่นที่ไม่เกี่ยวข้อง

กฎของอนุญาโตตุลาการในทุกประเด็นยกเว้นว่าศาลมีอำนาจแต่เพียงผู้เดียว: (i) ในการตัดสินใจเกี่ยวกับการอนุญาโตตุลาการ เช่นเดียวกับการก่อตั้ง การมีอยู่ ขอบเขต ความถูกต้อง และการบังคับใช้ของข้อตกลงอนุญาโตตุลาการนี้; (ii) เพื่อตัดสินใจว่าคู่สัญญาทั้งสองฝ่ายได้ปฏิบัติตามข้อกำหนดก่อนการอนุญาโตตุลาการหรือไม่ (รวมถึงหนังสือแจ้งข้อพิพาทและแบบฟอร์มการเรียกร้องอนุญาโตตุลาการที่เป็นรายบุคคล) (iii) เพื่อบังคับใช้ข้อห้ามในกลุ่ม ตัวแทน อัยการสูงสุดเอกชน หรือการดำเนินการหรือการพิจารณาคดีแบบผสมผสาน หรือการบรรเทาทุกข์ตามคำสั่งสาธารณะ และ (iv) สั่งให้อนุญาโตตุลาการดำเนินการหากว่าไม่ปฏิบัติตามข้อตกลงนี้

หากว่าการแจ้งข้อพิพาทของคุณเกี่ยวข้องกับการเรียกร้องที่คล้ายคลึงกับลูกค้ารายอื่นอย่างน้อย 24 ราย และหากว่าคุณและลูกค้ารายอื่น ๆ เหล่านั้นเป็นตัวแทนจากทนายความคนเดียวกัน หรือโดยทนายความที่ประสานงานระหว่างกัน คุณและเราตกลงว่าการเรียกร้องสิทธิ์เหล่านี้จะเป็น “คดีที่เกี่ยวข้อง” คุณและเราตกลงว่ากฎเพิ่มเติมสำหรับการอนุญาโตตุลาการโดยรวมของ AAA ที่มีผลใช้บังคับเมื่อมีการยื่นคำร้องอนุญาโตตุลาการ ตามที่แก้ไขโดยส่วนที่ 2.14 นี้จะนำไปใช้กับคดีที่เกี่ยวข้อง กรณีที่เกี่ยวข้องสามารถยื่นเป็นชุดได้สูงสุด 50 อนุญาโตตุลาการต่อครั้งเท่านั้น และอนุญาโตตุลาการแต่ละรายการจะได้รับการแก้ไขในลักษณะต่อไปนี้: (i) สำหรับชุดแรก แต่ละฝ่ายสามารถเลือกได้สูงสุด 25 คดีจากกรณีที่เกี่ยวข้องเหล่านี้ จะถูกยื่นและแก้ไขในการอนุญาโตตุลาการรายบุคคลภายใต้มาตรา 2.14 นี้ (ii) ไม่มีการยื่นหรือดำเนินคดีในคดีที่เกี่ยวข้องอื่นใดในอนุญาโตตุลาการจนกว่าอนุญาโตตุลาการชุดแรกสูงสุด 50 คดีจะได้รับการแก้ไข และ (iii) หากว่าหลังจากชุดแรกนั้น คู่สัญญาทั้งสองฝ่ายไม่สามารถแก้ไขคดีที่เกี่ยวข้องที่เหลือได้อย่างไม่เป็นทางการ อาจมีการยื่นคดีที่เกี่ยวข้องชุดที่สอง โดยแต่ละฝ่ายสามารถเลือกคดีที่เกี่ยวข้องได้มากถึง 25 คดีเพื่อแก้ไขในการอนุญาโตตุลาการแต่ละรายการภายใต้มาตรา 2.14 นี้ กระบวนการอนุญาโตตุลาการแยกเป็นชุดนี้จะดำเนินต่อไปจนกว่าทั้งสองฝ่ายจะแก้ไขคดีที่เกี่ยวข้องทั้งหมดอย่างไม่เป็นทางการหรือผ่านอนุญาโตตุลาการรายบุคคล ศาลมีอำนาจแต่เพียงผู้เดียวในการบังคับใช้ย่อหน้านี้ รวมถึงไม่ว่าจะนำไปใช้กับชุดการเรียกร้องสิทธิ์ที่กำหนดหรือไม่ และสั่งการให้ยื่นหรือดำเนินคดีอนุญาโตตุลาการที่ไม่ปฏิบัติตามย่อหน้านี้

(e)         ค่าธรรมเนียมและการชำระเงินในกระบวนการอนุญาโตตุลาการ

. กฎของ AAA จะใช้บังคับกับการชำระค่าธรรมเนียมการยื่นฟ้อง รวมทั้งค่าธรรมเนียมและค่าใช้จ่ายของ AAA และอนุญาโตตุลาการ

นอกจากนี้ ถ้าหากว่าข้อพิพาทไม่ได้เป็นส่วนหนึ่งของคดีที่เกี่ยวข้องที่อธิบายไว้ข้างต้น ข้อพิพาทดังกล่าวมีมูลค่าน้อยกว่า 75,000 ดอลลาร์สหรัฐฯ และก่อนที่จะเริ่มการอนุญาโตตุลาการ ได้ปฏิบัติตามข้อกำหนดก่อนการอนุญาโตตุลาการทั้งหมดในส่วนที่ 2.14 นี้ จากนั้น (1) เราจะคืนเงินค่าธรรมเนียมการยื่นฟ้องของคุณและชำระค่าธรรมเนียมและค่าใช้จ่ายของ AAA และอนุญาโตตุลาการโดยทันที และ (2) ถ้าหากว่าอนุญาโตตุลาการตัดสินให้คุณมากกว่าข้อเสนอที่เป็นลายลักษณ์อักษรครั้งล่าสุดของเราก่อนที่จะแต่งตั้งอนุญาโตตุลาการ เราจะจ่ายเงินให้คุณ (i) จำนวนเงินรางวัลหรือ 1,000 ดอลลาร์สหรัฐฯ (แล้วแต่จำนวนใดจะมากกว่า) และ (ii) สำหรับค่าธรรมเนียมทนายความที่สมเหตุสมผลใด ๆ ที่คุณเกิดขึ้น; และค่าใช้จ่ายที่เหมาะสมใด ๆ (รวมถึงค่าธรรมเนียมและค่าใช้จ่ายพยานผู้เชี่ยวชาญ) ที่ทนายความของคุณเกิดขึ้นจากการอนุญาโตตุลาการส่วนบุคคลของคุณ

(f)         การแยกข้อตกลงออกเป็นส่วน

. ถ้าหากว่าหลังจากการอุทธรณ์ทั้งหมดสิ้นสุดลง หากว่าศาลพบว่าส่วนหนึ่งส่วนใดของส่วนที่ 2.14 นี้ไม่สามารถบังคับใช้ได้สำหรับการเรียกร้องสิทธิ์หรือการร้องขอการเยียวยาใด ๆ คู่สัญญาทั้งสองฝ่ายตกลงที่จะอนุญาโตตุลาการการเรียกร้องสิทธิ์และการเยียวยาทั้งหมดภายใต้อนุญาโตตุลาการก่อนที่จะดำเนินคดีในศาลการเรียกร้องหรือการเยียวยาที่เหลืออยู่ (เช่น คำร้องขอให้มีการเยียวยาโดยคำสั่งสาธารณะ ซึ่งในกรณีนี้อนุญาโตตุลาการจะออกคำชี้ขาดเกี่ยวกับความรับผิดและการบรรเทาทุกข์ส่วนบุคคลก่อนที่ศาลจะพิจารณาคำขอนั้น) มิฉะนั้น หากพบว่าส่วนอื่นใดของมาตรา 2.14 ไม่สามารถบังคับใช้ได้ ส่วนที่เหลือจะยังคงมีผลใช้บังคับ (โดยจะมีการออกคำชี้ขาดของอนุญาโตตุลาการก่อนที่การพิจารณาคดีของศาลจะเริ่มขึ้น)

2.15.      สัญญาเบ็ดเสร็จ

. ข้อตกลงนี้ (ร่วมกับเงื่อนไขการอนุญาตให้ใช้สิทธิฉบับพิมพ์หรือเงื่อนไขอื่น ๆ ที่มาพร้อมส่วนประกอบเพิ่มเติม โปรแกรมปรับปรุง และบริการต่าง ๆ ของซอฟต์แวร์ที่เราจัดหา และที่คุณเป็นผู้ใช้) และเงื่อนไขที่มีอยู่ในการเชื่อมโยงบนเว็บซึ่งระบุในข้อตกลงนี้ จะเป็นข้อตกลงฉบับเดียวสำหรับซอฟต์แวร์ รวมทั้งส่วนประกอบเพิ่มเติม โปรแกรมปรับปรุง และบริการต่างๆ เหล่านี้ (เว้นแต่ว่าเราจะจัดให้มีเงื่อนไขอื่น ๆ เกี่ยวกับส่วนประกอบเพิ่มเติม โปรแกรมปรับปรุง หรือบริการต่าง ๆ ดังกล่าว)

2.16.      กฎหมายที่ใช้บังคับและสถานที่ในการระงับข้อพิพาท

. หากคุณซื้อหรือได้รับซอฟต์แวร์มาจากในประเทศสหรัฐอเมริกาหรือแคนาดา ให้นำกฎหมายประจำมลรัฐ หรือมณฑลที่คุณอาศัยอยู่ (หรือหากเป็นธุรกิจ ที่สำนักงานหลักของธุรกิจของคุณตั้งอยู่) มาใช้กำกับควบคุมหรือกับการตีความข้อตกลงฉบับนี้ การเรียกร้องให้ชดเชยการผิดสัญญา และการเรียกร้องอื่น ๆ นอกเหนือจากนี้ (รวมถึง การปกป้องผู้บริโภค การแข่งขันอย่างไม่เป็นธรรม และการเรียกร้องจากการถูกละเมิด) โดยไม่คำนึงว่าจะมีความขัดแย้งในข้อกฎหมายหรือไม่ก็ตาม ยกเว้นกรณีที่ FAA ได้เข้ามากำกับดูแลทุกเรื่องที่เกี่ยวกับกระบวนการอนุญาโตตุลาการ หากคุณได้รับซอฟต์แวร์มาจากประเทศอื่น กฎหมายของประเทศนั้นจะมีผลใช้บังคับ เว้นแต่ FAA จะใช้บังคับกับทุกประเด็นที่เกี่ยวข้องกับการอนุญาโตตุลาการ หากอยู่ในขอบเขตอำนาจพิจารณาคดีของรัฐบาลกลางสหรัฐฯ คุณและ Microsoft ยินยอมรับเขตอำนาจศาลและสถานที่พิจารณาคดีเฉพาะของศาลรัฐบาลกลาง (federal court) ประจำคิงเคาวน์ตี้ มลรัฐวอชิงตัน เพื่อไต่สวนข้อพิพาททั้งหมดที่ไม่ได้พิจารณาในการอนุญาโตตุลาการหรือศาลคดีความขนาดเล็ก หากมิฉะนั้น คุณและ Microsoft ยินยอมรับเขตอำนาจศาลและสถานที่พิจารณาคดีเฉพาะของศาลสูง (Superior Court) ประจำคิงเคาวน์ตี้ มลรัฐวอชิงตัน เพื่อไต่สวนข้อพิพาททั้งหมดที่ไม่ได้พิจารณาในการอนุญาโตตุลาการหรือศาลคดีความขนาดเล็ก

2.17.      สิทธิของผู้บริโภคที่จะแตกต่างกันไปในแต่ละภูมิภาค

. ข้อตกลงฉบับนี้อธิบายถึงสิทธิทางกฎหมายบางประการ คุณอาจมีสิทธิ์อื่น ซึ่งรวมถึงสิทธิของผู้บริโภคภายใต้กฎหมายในรัฐ จังหวัด หรือประเทศของคุณ คุณยังอาจมีสิทธิตามผู้ที่คุณได้รับซอฟต์แวร์มา ข้อตกลงฉบับนี้ไม่ได้เปลี่ยนแปลงสิทธิ์อื่น ๆ เหล่านั้นของคุณ หากกฎหมายในรัฐ จังหวัด หรือประเทศของคุณไม่อนุญาตให้ทำเช่นนั้น ยกตัวอย่างเช่น หากว่าคุณซื้อหรือได้รับซอฟต์แวร์มาจากหนึ่งในภูมิภาคด้านล่างนี้ หรือมีการบังคับใช้กฎหมายของประเทศ ข้อกำหนดดังต่อไปนี้มีผลบังคับใช้กับคุณ:

(a)         ออสเตรเลีย

. สำหรับผู้บริโภคที่อาศัยอยู่ในประเทศออสเตรเลีย บริการของเรามาพร้อมกับการรับประกันที่ไม่สามารถยกเว้นได้ภายใต้กฎหมายคุ้มครองผู้บริโภคของประเทศออสเตรเลีย สำหรับความชำรุดบกพร่องที่สำคัญกับบริการ คุณมีสิทธิ:

(i)         ในการยกเลิกสัญญาการบริการของคุณกับเรา และ

(ii)         ในการขอคืนเงินส่วนที่ไม่ได้ใช้ หรือเพื่อชดเชยมูลค่าที่ลดลง

คุณมีสิทธิที่จะได้รับการชดเชยสำหรับการขาดทุนหรือความเสียหายที่คาดการณ์ได้อื่นใดอย่างสมเหตุสมผล หากความชำรุดบกพร่องไม่ใช่ความชำรุดบกพร่องที่สำคัญ คุณมีสิทธิที่จะแก้ไขปัญหากับบริการได้ภายในระยะเวลาที่เหมาะสม และหากไม่ดำเนินการดังกล่าว คุณมีสิทธิในการยกเลิกสัญญาและขอรับเงินคืนสำหรับส่วนที่ไม่ได้ใช้ของสัญญา

(b)         เยอรมนีและออสเตรีย

. สำหรับผู้บริโภคที่อาศัยอยู่ในประเทศออสเตรเลีย บริการของเรามาพร้อมกับการรับประกันที่ไม่สามารถยกเว้นได้ภายใต้กฎหมายคุ้มครองผู้บริโภคของประเทศออสเตรเลีย สำหรับความชำรุดบกพร่องที่สำคัญกับบริการ คุณมีสิทธิ:

(i)         ไม่มีการรับประกันตามสัญญา . Microsoft จะไม่ให้การรับประกันตามสัญญาแก่ซอฟต์แวร์ที่ได้รับลิขสิทธิ์ถูกต้อง

(ii)         การจำกัดความรับผิด . ในกรณีของการประพฤติผิดโดยเจตนา หรือประมาทเลินเล่ออย่างร้ายแรง การเรียกร้องค่าเสียหายตามกฎหมาย Product Liability Act รวมถึงกรณีการเสียชีวิต หรือเป็นเหตุให้บุคคลหรือร่างกายได้รับบาดเจ็บ Microsoft จะต้องรับผิดตามกฎหมายที่เป็นลายลักษณ์อักษร

โดยอยู่ภายใต้ประโยคก่อนหน้านี้ Microsoft จะต้องรับผิดในความประมาทเลินเล่อเล็กน้อยเฉพาะกรณีที่ Microsoft ละเมิดหน้าที่ตามสัญญาในสาระสำคัญดังกล่าว ซึ่งการทำตามหน้าที่นี้จะช่วยในการดำเนินการที่ถูกต้องของข้อตกลงนี้ การละเมิดหน้าที่นี้จะเป็นอันตรายต่อวัตถุประสงค์ของข้อตกลงนี้และการปฏิบัติตามหน้าที่นี้ซึ่งคู่สัญญาอาจไว้ใจโดยเสมอ (หรือที่เรียกกันว่า "Cardinal Obligation") ในกรณีของการประมาทเลินเล่อที่ไม่ร้ายแรงอื่น ๆ Microsoft จะไม่รับผิดจากการประมาทเลินเล่อเพียงเล็กน้อย

2.18.      คำปฏิเสธการรับประกัน

. ซอฟต์แวร์นี้เป็นการอนุญาตให้ใช้สิทธิ "ตามที่เป็น" "ภายใต้ข้อบกพร่องทั้งหมด" และ "ตามที่มี" คุณเป็นผู้รับความเสี่ยงในการใช้งานเอง MICROSOFT ไม่ได้ให้การรับประกัน การรับรอง หรือเงื่อนไขที่แสดงอย่างชัดแจ้งอื่นใดอีก คุณอาจมีสิทธิเพิ่มเติมของผู้บริโภคหรือการรับประกันตามกฎหมายภายใต้กฎหมายท้องถิ่นของคุณ ซึ่งข้อตกลงฉบับนี้ไม่สามารถเปลี่ยนแปลงได้ MICROSOFT ละเว้นการรับประกันโดยนัยทั้งหมด ซึ่งรวมถึงความสามารถในเชิงพาณิชย์ ความสมบูรณ์เพื่อการใช้งานเฉพาะด้าน และการไม่ละเมิด ตามขอบเขตที่อนุญาตให้ใช้ตามกฎหมายท้องถิ่นของคุณ

2.19.      ข้อจำกัดและข้อยกเว้นด้านการแก้ไขเยียวยาและความเสียหาย

. หากคุณมีมูลเหตุที่จะเรียกร้องค่าชดเชยความเสียหาย ถึงแม้จะมีถ้อยคำปฏิเสธความรับผิดชอบในการรับประกันดังที่กล่าวมาก่อนหน้านี้ก็ตาม คุณสามารถเรียกร้องค่าชดเชยจาก MICROSOFT และซัพพลายเออร์ของ MICROSOFT ได้เฉพาะค่าความเสียหายโดยตรงตามจำนวนที่คุณจ่ายซื้อซอฟต์แวร์หรือ 5.00 ดอลลาร์สหรัฐฯ ขึ้นอยู่กับจำนวนใดจะมากกว่า คุณไม่สามารถเรียกร้องความเสียหายอื่นใดที่เกิดขึ้น รวมถึงความเสียหายอันเป็นผลสืบเนื่อง การสูญเสียผลกำไร ความเสียหายพิเศษ ความเสียหายโดยอ้อม หรือความเสียหายอันเกิดจากการผิดสัญญา เท่าที่กฎหมายอนุญาตไว้สูงสุด ข้อจำกัดและข้อยกเว้นเหล่านี้จะใช้บังคับกับประเด็นใด ๆ หรือการเรียกร้องสิทธิใด ๆ ที่เกี่ยวข้องกับซอฟต์แวร์

MICROSOFT સૉફ્ટવેરના લાઇસન્સની શરતો

MICROSOFT EDGE

ઇન્સ્ટોલેશન પછી, આ શરતો Microsoft Edgeમાં edge://terms પર જોઈ શકાય છે.

જો તમે યુનાઇટેડ સ્ટેટમાં રહેતા હોવ (અથવા તે વ્યવસાયનું મુખ્ય સ્થાન હોય), તો નીચે "બાઇંડિંગ આર્બિટ્રેશન અને ક્લાસ એક્શન વેવર" અનુભાગ 2.14 વિશે વાંચો. વિવાદો કેવી રીતે ઉકેલવામાં આવે છે તેને તે પ્રભાવિત કરે છે.

આ તમારી અને Microsoft Corporation (અથવા તેના સહયોગીઓમાંની એક) વચ્ચે લાઇસન્સ સંબંધી સમજૂતી છે જે તમારા અધિકારો, જવાબદારીઓ અને શરતોનું વર્ણન કરે છે કે જેના પર તમે સૉફ્ટવેરનો ઉપયોગ કરી શકો છો અને કોઈપણ અપડેટ્સ ("સોફ્ટવેર") (તે હદ સુધી સિવાય કે અપડેટ્સ નવા સાથે જોડાયેલા હોય છે અથવા વધારાની શરતો, જે કિસ્સામાં તે વિવિધ શરતો સંભવિત રીતે લાગુ પડે છે અને પહેલેથી અપડેટ કરેલ સૉફ્ટવેરને લગતા તમારા અથવા Microsoftના અધિકારોમાં ફેરફાર કરશે નહીં ). તમારે કોઈ પણ લિંક્ડ શરતો સહિત સમગ્ર સમજૂતીની સમીક્ષા કરવી જોઈએ, કારણ કે તમામ શરતો મહત્વપૂર્ણ છે અને સાથે મળીને આ સમજૂતી બનાવો જે તમને લાગુ પડે.

આ સમજૂતીનો સ્વીકાર કરીને અથવા સૉફ્ટવેર ડાઉનલોડ કરીને અથવા તેનો ઉપયોગ કરીને, તમે આ બધી શરતો સાથે સંમત થાઓ છો અને તમારા Windows લાઇસન્સની શરતોમાં અથવા નીચે અનુભાગ 2.7માં વર્ણવેલ ગોપનીયતા નિવેદન અનુસાર સૉફ્ટવેરના તમારા ઉપયોગ દરમિયાન ચોક્કસ માહિતીના પ્રસારણ માટે સંમતિ આપો છો. જો તમે આ શરતોને સ્વીકારો અને તેની સાથે અનુસરતા ન હો તો તમે સૉફ્ટવેર અથવા તેની વિશેષતાઓનો ઉપયોગ કરી શકશો નહીં.

આ સમજૂતીમાં બે વિભાગનો સમાવેશ થાય છે: જો તમે કોઈ પણ Windows પ્લેટફોર્મ પર સૉફ્ટવેરનો ઉપયોગ કરો છો તો અનુભાગ 1 લાગુ પડે છે; જો તમે MacOS, iOS, Android અને Linux સહિતના કોઇ પણ નોન-Windows પ્લેટફોર્મ પર તેનો ઉપયોગ કરો તો અનુભાગ 2 લાગુ પડે છે.

1.     WINDOWS ડિવાઇસેસ માટે MICROSOFT EDGE

1.1.        Windows લાઇસન્સની શરતો લાગુ પડે છે

. તમારા Microsoft સૉફ્ટવેરના લાઇસન્સની શરતો, Windows ઑપરેટિંગ સિસ્ટમ ("Windows લાઇસન્સની શરતો") કોઈ પણ Windows પ્લેટફોર્મ પર આ સૉફ્ટવેરના તમારા ઉપયોગને નિયંત્રિત કરે છે અને આ સમજૂતીનો ભાગ છે. તમે આના પર જઈને Windows લાઇસન્સની શરતો જોઈ શકો છો https://aka.ms/useterms અથવા સેટિંગ્સ - સિસ્ટમ - Windowsની અંદર જઈ રહ્યા છીએ.

1.1.1.        અપડેટ્સ

. સૉફ્ટવેર સમયાંતરે અપડેટ માટે તપાસ કરી શકે છે અને તમારા માટે તેને ડાઉનલોડ અને ઇન્સ્ટોલ કરી શકે છે. તમે ફક્ત Microsoft અથવા અધિકૃત સ્રોતોથી જ અપડેટ્સ મેળવી શકો છો, અને Microsoftને તમને તે અપડેટ્સ પ્રદાન કરવા માટે તમારી સિસ્ટમને અપડેટ કરવાની જરૂર પડી શકે છે તમને તે અપડેટ્સ પ્રદાન કરવા માટે તમારી સિસ્ટમ. આ કરાર સ્વીકારીને અથવા સૉફ્ટવેરનો ઉપયોગ કરીને, તમે કોઈપણ વધારાની સૂચના વિના આ પ્રકારના સ્વયંચાલિત અપડેટ્સ પ્રાપ્ત કરવા માટે સંમતિ આપો છો.

1.2.        પૂર્વાવલોકનો

. તમે પૂર્વાવલોકન, આંતરિક, બીટા, અથવા સૉફ્ટવેર ("પૂર્વાવલોકનો")ના અન્ય પૂર્વ-પ્રકાશન સંસ્કરણોનો ઉપયોગ કરવાનું પસંદ કરી શકો છો, જે Microsoft ઉપલબ્ધ કરાવી શકે છે. તમે સૉફ્ટવેરની સમાપ્તિની તારીખ સુધી જ પૂર્વાવલોકનોનો ઉપયોગ કરી શકો છો અને જ્યાં સુધી તમે આ કરારની બધી શરતોનું અનુપાલન કરો ત્યાં સુધી. પૂર્વાવલોકનો પ્રાયોગિક છે અને વ્યાવસાયિક રૂપે પ્રકાશિત સંસ્કરણથી નોંધપાત્ર રીતે અલગ હોઈ શકે છે. તમારી Windows લાઇસન્સની શરતોમાં તેનાથી વિપરીત કંઈ પણ હોવા છતાં, પૂર્વાવલોકનો "AS IS" પ્રદાન કરવામાં આવે છે, અને આ સંસ્કરણોને કોઈ વૉરંટી, ગર્ભિત અથવા સ્પષ્ટ (મર્યાદિત વૉરંટી સહિત) લાગુ પડતી નથી. તમારા ડિવાઇસ પર પૂર્વાવલોકનો ઇન્સ્ટોલ કરીને, તમે તમારા ડિવાઇસની વૉરંટીને રદબાતલ કરી શકો છો અથવા પ્રભાવિત કરી શકો છો અને જો લાગુ પડતું હોય તો, તમારા ડિવાઇસના ઉત્પાદક અથવા નેટવર્ક ઑપરેટર તરફથી સપોર્ટ પામવા માટે હકદાર ન પણ હોઈ શકો. Microsoft તમને થયેલા કોઈપણ નુકસાન માટે જવાબદાર નથી. Microsoft પ્રિવ્યુ માટે સમર્થન સેવાઓ આપી શકશે નહીં. જો તમે પૂર્વાવલોકન ("સબમિશન") વિશે Microsoftની ટિપ્પણીઓ, સૂચનો અથવા અન્ય પ્રતિસાદ પ્રદાન કરો છો, તો તમે Microsoft અને તેના ભાગીદારોને કોઈ પણ રીતે અને કોઈ પણ હેતુ માટે સબમિશનનો ઉપયોગ કરવાનો અધિકાર આપો છો.

1.2.1.        પૂર્વાવલોકનો માટે ડેટા સંગ્રહ

. પૂર્વાવલોકનો માટે, ગોપનીયતા અને ફીચર સેટિંગ્સ ધાર્યા પ્રમાણે કામ ન પણ કરી શકે, અને પૂર્વાવલોકનો અન્ય Windows ગોપનીયતા સૅટિંગ્ઝ સાથે કામ ન પણ કરી શકે, જેમાં Windows 10 અથવા Windows 11 માટે ડાયગ્નોસ્ટિક ડેટા સેટિંગ્સનો પણ સમાવેશ થાય છે. પૂર્વાવલોકનોના તમારા ઉપયોગમાંથી એકત્રિત કરવામાં આવેલી માહિતી, જેમાં નિદાન, તકનીકી, ભૂલ સંબંધી રિપોર્ટ્સ, ક્રેશ ડમ્પ્સ અને અન્ય સંબંધિત ડેટાનો સમાવેશ થાય છે, જેમાં પૂર્વાવલોકનો ચાલી રહ્યા છે, Windows, આ સૉફ્ટવેર અને પૂર્વાવલોકનોને અદ્યતન, સુરક્ષિત અને યોગ્ય રીતે સંચાલિત રાખવા માટે ઉપયોગ, સંગ્રહિત, પ્રક્રિયા અને વિશ્લેષણ કરી શકાય છે. તે અમને Microsoft ઉત્પાદનો અને સેવાઓ સુધારવામાં પણ મદદ કરે છે અને Microsoft ગોપનીયતા નિવેદનમાં વર્ણવેલ અન્ય હેતુ માટે તેનો ઉપયોગ થઈ શકે છે. જો તમે પૂર્વાવલોકનોમાં ઉપલબ્ધ નિયંત્રણો મારફતે ડેટા એકત્રીકરણને અક્ષમ કરો છો, તો અમે Microsoft ગોપનીયતા સ્ટેટમેન્ટના Windows ડાયગ્નોસ્ટિકસ વિભાગમાં વર્ણવ્યા પ્રમાણે ડાઉનલોડ, ઇન્સ્ટોલેશન અને પૂર્વાવલોકનો અને "મૂળભૂત" ડેટા વિશે ડાયગ્નોસ્ટિક માહિતી એકત્રિત કરવાનું ચાલુ રાખી શકીએ છીએ https://aka.ms/privacy.

1.3.        અન્ય સેવાઓ

. સૉફ્ટવેરમાં એવી એપ્લિકેશનો અથવા ફીચર્સનો સમાવેશ થઈ શકે છે જે ઑનલાઇન સેવાઓ માટે ઍક્સેસ પોઇન્ટ પૂરો પાડે છે, અથવા તેના પર આધાર રાખે છે, અને તે સેવાઓનો ઉપયોગ કેટલીકવાર અલગ નિયમો અને ગોપનીયતા નીતિઓ દ્વારા સંચાલિત થાય છે, જેમ કે Microsoft સેવા કરાર https://aka.ms/msa. લાગુ પડતું હોય તે પ્રમાણે તમે સેવાની ઉપયોગની શરતો અથવા એપ્લિકેશનના સૅટિંગ્ઝને જોઈને આ નિયમો અને નીતિઓ જોઈ શકો છો.. સેવાઓ બધા પ્રદેશોમાં ઉપલબ્ધ હોઈ શકતી નથી. તમે સૉફ્ટવેરથી અલગ Microsoft Azure સેવામાં કૉલ કરવા માટે સૉફ્ટવેર ઉપયોગ કરી રહ્યું હોય તેવા ટોકનનો ઉપયોગ ન કરી શકો તેમ બની શકે છે.

2.     બિન-WINDOWS ડિવાઇસેસ માટે MICROSOFT EDGE

2.1.        ઇન્સ્ટોલેશન અને વપરાશ સંબંધી અધિકારો

. સોફ્ટવેરને લાઇસન્સ આપવામાં આવ્યું છે, ન કે વેચવામાં આવ્યું છે. આ કરાર હેઠળ, અમે તમને તમારા ડિવાઇસ પર સૉફ્ટવેરનો એક દાખલો ઇન્સ્ટોલ કરવાનો અને ચલાવવાનો અધિકાર આપીએ છીએ, જેનો ઉપયોગ એક સમયે એક વ્યક્તિ દ્વારા કરવામાં આવે છે, જે MacOS, iOS, Android અને Linux સહિત બિન-Windows પ્લેટફોર્મ ચલાવે છે, જ્યાં સુધી તમે આ કરારમાં સમાવિષ્ટ શરતો અને પ્રતિબંધોનું અનુપાલન કરો છો.

2.2.        તૃતીય પક્ષનું સૉફ્ટવેર

2.3.        પૂર્વાવલોકનો

. તમે પૂર્વાવલોકન, આંતરિક, બીટા, અથવા સૉફ્ટવેર ("પૂર્વાવલોકનો")ના અન્ય પૂર્વ-પ્રકાશન સંસ્કરણોનો ઉપયોગ કરવાનું પસંદ કરી શકો છો, જે Microsoft ઉપલબ્ધ કરાવી શકે છે. તમે સૉફ્ટવેરની સમાપ્તિની તારીખ સુધી જ પૂર્વાવલોકનોનો ઉપયોગ કરી શકો છો અને જ્યાં સુધી તમે આ કરારની બધી શરતોનું અનુપાલન કરો ત્યાં સુધી. પૂર્વાવલોકનો પ્રાયોગિક છે અને વ્યાવસાયિક રૂપે પ્રકાશિત સંસ્કરણથી નોંધપાત્ર રીતે અલગ હોઈ શકે છે. પૂર્વાવલોકનો "AS IS" પ્રદાન કરવામાં આવે છે, અને આ સંસ્કરણોને કોઈ વૉરંટી, ગર્ભિત અથવા સ્પષ્ટ (મર્યાદિત વૉરંટી સહિત) લાગુ પડતી નથી. તમારા ડિવાઇસ પર પૂર્વાવલોકનો ઇન્સ્ટોલ કરીને, તમે તમારા ડિવાઇસની વૉરંટીને રદબાતલ કરી શકો છો અથવા પ્રભાવિત કરી શકો છો અને જો લાગુ પડતું હોય તો, તમારા ડિવાઇસના ઉત્પાદક અથવા નેટવર્ક ઑપરેટર તરફથી સપોર્ટ પામવા માટે હકદાર ન પણ હોઈ શકો. Microsoft તમને થયેલા કોઈપણ નુકસાન માટે જવાબદાર નથી. Microsoft પૂર્વાવલોકન માટે સમર્થન સેવાઓ આપી શકશે નહીં. જો તમે પૂર્વાવલોકન ("સબમિશન") વિશે Microsoftની ટિપ્પણીઓ, સૂચનો અથવા અન્ય પ્રતિસાદ પ્રદાન કરો છો, તો તમે Microsoft અને તેના ભાગીદારોને કોઈ પણ રીતે અને કોઈ પણ હેતુ માટે સબમિશનનો ઉપયોગ કરવાનો અધિકાર આપો છો.

2.3.1.        પૂર્વાવલોકનો માટે ડેટા સંગ્રહ

. અનુભાગ 2.3 હેઠળ આવરી લેવામાં આવેલ પૂર્વાવલોકન માટે, ગોપનીયતા અને સુવિધા સેટિંગ્સ હેતુ મુજબ કાર્ય કરી શકશે નહીં, અને પૂર્વાવલોકન અન્ય ઓપરેટિંગ સિસ્ટમની ગોપનીયતા સેટિંગ્સ સાથે કામ કરશે નહીં. પૂર્વાવલોકનને અદ્યતિત, સલામત અને યોગ્ય રીતે ઓપરેટ થાય તે રીતે રાખવા માટે, તમારા ઉપકરણોના નૈદાનિક, તકનીકી, ત્રુટી રીપોર્ટ, ક્રેશ ડમ્પ અને અન્ય સંબંધિત ડેટા સહિત તમારા પ્રિવ્યુના ઉપયોગમાંથી સંગ્રહિત ડેટાનો ઉપયોગ, સંગ્રહ થઇ શકે છે, પ્રક્રિયા અને વિશ્લેષણ કરવામાં આવી શકે છે. તે અમને Microsoft ઉત્પાદનો અને સેવાઓ સુધારવામાં પણ મદદ કરે છે અને Microsoft ગોપનીયતા નિવેદનમાં વર્ણવેલ અન્ય હેતુ માટે તેનો ઉપયોગ થઈ શકે છે. જો તમે પૂર્વાવલોકનોમાં ઉપલબ્ધ નિયંત્રણો દ્વારા ડેટા સંગ્રહને અક્ષમ કરો છો, તો અમે Microsoft ગોપનીયતા નિવેદનના Windows ડાયગ્નોસ્ટિકસ વિભાગમાં વર્ણવ્યા મુજબ ડાઉનલોડ, ઇન્સ્ટોલેશન અને કોઈ પણ પૂર્વાવલોકન દૂર કરવા અને "મૂળભૂત" ડેટા વિશે નૈદાનિક માહિતી એકત્રિત કરવાનું ચાલુ રાખી શકીએ છીએ. https://aka.ms/privacy.

2.4.        અન્ય સેવાઓ

. સૉફ્ટવેરમાં એવી એપ્લિકેશનો અથવા ફીચર્સનો સમાવેશ થઈ શકે છે જે ઑનલાઇન સેવાઓ માટે એક્સેસ પોઇન્ટ પૂરો પાડે છે, અથવા તેના પર આધાર રાખે છે, અને તે સેવાઓનો ઉપયોગ કેટલીકવાર અલગ નિયમો અને ગોપનીયતા નીતિઓ દ્વારા સંચાલિત થાય છે, જેમ કે Microsoft સેવા કરાર https://aka.ms/msa પર. લાગુ પડતું હોય તે પ્રમાણે તમે સેવાની ઉપયોગની શરતો અથવા એપ્લિકેશનના સેટિંગ્ઝને જોઈને આ નિયમો અને નીતિઓ જોઈ શકો છો.. સેવાઓ બધા પ્રદેશોમાં ઉપલબ્ધ હોઈ શકતી નથી.

2.5.        તમારી સાથે સંચાર

2.6.        પ્રતિક્રિયા

2.7.         ગોપનીયતાy; ડેટાના ઉપયોગ માટે સંમતિ

AAA તેના વાણિજ્યક આર્બિટ્રેશન નિયમો (Commercial Arbitration Rules) (અથવા જો તમે વ્યક્તિ છો અને વ્યક્તિગત અથવા ઘરના ઉપયોગ માટે સેવાઓનો ઉપયોગ કરો છો, અથવા જો તમે વ્યક્તિ હોવ કે ના હોવ, વિવાદનું મૂલ્ય $75,000 USD અથવા તેથી ઓછું હોય અથવા તમે કેવી રીતે સેવાનો, તેના ગ્રાહક આર્બિટ્રેશન નિયમોનો (Consumer Arbitration Rules) ઉપયોગ કરો છો) હેઠળ કોઈપણ આર્બિટ્રેશન હાથ ધરશે. AAAના સામૂહિક આર્બિટ્રેશન પૂરક નિયમો, આ અનુભાગ 2.14 દ્વારા સંશોધિત કર્યા મુજબ, સંબંધિત કેસોમાં લાગુ પડશે (નીચે ચર્ચા કરવામાં આવી છે). વધુ માહિતી માટે જુઓ https://aka.ms/adr. આ કરાર એના કોઈ પણ લાગુ પડતા નિયમો સાથે ઘર્ષણમાં આવે તે હદ સુધી સંચાલિત થાય છે. આર્બિટ્રેશનની પ્રક્રિયા શરૂ કરવા માટે, આના પર ઉપલબ્ધ આર્બિટ્રેશન ફૉર્મની માંગ સબમિટ કરો https://go.microsoft.com/fwlink/?LinkId=245497 પર AAAને સબમિટ કરો અને અમને એક કૉપિ મેઇલ કરો. આ ફૉર્મમમાં એવી માહિતી હોવી જોઈએ જે તમારા અને તમારા દાવા માટે વિશિષ્ટ હોય. $25,000 USD અથવા તેથી ઓછા ડોલરના વિવાદમાં, કોઈપણ સુનાવણી ટેલિફોન પર હશે સિવાય કે આર્બિટ્રેટરને વ્યક્તિગત રૂપે સુનાવણી કરવાનું સારું કારણ મળે. કોઈ પણ વ્યક્તિગત સુનાવણી તમારા નિવાસસ્થાન અથવા કિંગ કાઉન્ટી, વોશિંગ્ટનની તમારા પ્રાદેશિક વિભાગમાં હાથ ધરવામા આવશે. આર્બિટ્રેટર વ્યક્તિગત રીતે તમારા માટે નુકસાનની ચુકવણીનો નિર્ણય અદાલત આપી શકે તેમ આપી શકે છે. આર્બિટ્રેટર તમારા વ્યક્તિગત દાવાને સંતોષવા માટે તમને વ્યક્તિગત રીતે જ ઘોષણાત્મક અથવા આદેશાત્મક રાહત આપી શકે છે, પરંતુ બિન-પક્ષકારોને અસર કરે તેવી રાહત નહીં.

આર્બિટ્રેટર તમામ સમસ્યાઓ પર નિયમો બનાવે છે સિવાય કે કોર્ટ પાસે વિશિષ્ટ સત્તા છેઃ (i) આર્બિટ્રેટરનો નિર્ણય લેવા માટે, તેમજ આ આર્બિટ્રેશન સમજૂતીની રચના, અસ્તિત્વ, અવકાશ, માન્યતા અને અમલીકરણની ક્ષમતા; (ii) એ નક્કી કરવું કે શું આ પક્ષકારોએ પૂર્વ-આર્બિટ્રેશનની જરૂરિયાતોનું પાલન કર્યું છે કે કેમ (વ્યક્તિગત રીતે વિવાદની નોટિસ અને આર્બિટ્રેટરનાં ફૉર્મ્સની માગ સહિત); (iii) વર્ગ, પ્રતિનિધિ, ખાનગી એટર્ની-જનરલ, અથવા સંયુક્ત પગલાં અથવા કાર્યવાહી, અથવા જાહેર મનાઈહુકમ રાહત પરના પ્રતિબંધનો અમલ કરવો; અને (iv) જો કોઈ આર્બિટ્રેટર આ સમજૂતીનું પાલન ન કરે તો તેને આગળ વધવાનો આદેશ આપવો.

જો તમારા વિવાદની નોટિસમાં ઓછામાં ઓછા 24 અન્ય ગ્રાહકો જેવા જ દાવાઓનો સમાવેશ થતો હોય, અને જો તમે અને તે અન્ય ગ્રાહકોનું પ્રતિનિધિત્વ એક જ વકીલો દ્વારા અથવા વકીલો દ્વારા કરવામાં આવે જેઓ એકબીજા સાથે સંકલન કરી રહ્યા હોય, તો તમે અને અમે સંમત થાઓ છો કે આ દાવાઓ "સંબંધિત કેસો" હશે. તમે અને અમે સંમત થાઓ છો કે જ્યારે આર્બિટ્રેશનની માગ દાખલ કરવામાં આવે ત્યારે અમલમાં રહેલા AAAના સામૂહિક આર્બિટ્રેશન પૂરક નિયમો, જેમ કે આ અનુભાગ 2.14 દ્વારા સંશોધિત કરવામાં આવ્યા છે, તે સંબંધિત કેસોને લાગુ પડશે. સંબંધિત કેસો એક સાથે 50 વ્યક્તિગત આર્બિટ્રેશન્સની બેચમાં જ દાખલ કરી શકાશે અને તે વ્યક્તિગત આર્બિટ્રેશન્સનું સમાધાન નીચેની રીતે થઈ શકશેઃ (i) પ્રથમ બેચ માટે, દરેક પક્ષ આ સંબંધિત કેસોમાંથી 25 સુધીની પસંદગી કરી શકે છે, જે આ અનુભાગ 2.14 હેઠળ વ્યક્તિગત મધ્યસ્થતાઓમાં દાખલ કરવામાં આવશે અને તેનું નિરાકરણ લાવી શકાશે; (ii) જ્યાં સુધી 50 સુધી વ્યક્તિગત આર્બિટ્રેશનની પ્રથમ બેચનો ઉકેલ ન આવે ત્યાં સુધી અન્ય કોઈ પણ સંબંધિત કેસ આર્બિટ્રેશનમાં દાખલ કરી શકાશે નહીં અથવા તેની સામે કાર્યવાહી કરી શકાશે નહીં; અને (iii) જો તે પ્રથમ બેચ પછી, પક્ષકારો બાકીના સંબંધિત કેસોને અનૌપચારિક રીતે ઉકેલવામાં અસમર્થ હોય, તો સંબંધિત કેસોની બીજી બેચ દાખલ કરી શકાય છે, જેમાં દરેક પક્ષ આ અનુભાગ 2.14 હેઠળ વ્યક્તિગત મધ્યસ્થતામાં ઉકેલવા માટે સંબંધિત કેસોમાંથી 25 સુધીની પસંદગી કરી શકે છે. બેચ્ડ વ્યક્તિગત આર્બિટ્રેશનની આ પ્રક્રિયા જ્યાં સુધી પક્ષકારો તમામ સંબંધિત કેસોને અનૌપચારિક રીતે અથવા વ્યક્તિગત આર્બિટ્રેશન્સ મારફતે ઉકેલે નહીં ત્યાં સુધી ચાલુ રહેશે. કોર્ટ પાસે આ ફકરાનો અમલ કરવાની વિશિષ્ટ સત્તા છે, જેમાં તે ચોક્કસ દાવાઓના સમૂહને લાગુ પડે છે કે કેમ અને આ ફકરાનું પાલન ન કરતા આર્બિટ્રેશનની ફાઇલિંગ અથવા કાર્યવાહીનો આદેશ આપવાનો સમાવેશ થાય છે.

(e)         આર્બિટ્રેશન ફી અને ચુકવણીઓ

. AAA ના નિયમો દાખલ કરવા માટેની ફી અને AAA અને આર્બિટ્રેટરની ફી અને ખર્ચની ચુકવણીનું સંચાલન કરશે.

વધુમાં, જો આ વિવાદ ઉપર વર્ણવેલા સંબંધિત કેસોનો ભાગ ન હોય, તો આ વિવાદમાં $75,000થી ઓછી રકમનો સમાવેશ થાય છે, અને આર્બિટ્રેશનની શરૂઆત કરતા પહેલા તમે આ અનુભાગ 2.14માં તમામ પૂર્વ-આર્બિટ્રેશનની આવશ્યકતાઓનું પાલન કર્યું હતું, તો (1) અમે તાત્કાલિક તમારી ફાઇલિંગ ફીનું વળતર આપીશું અને AAA અને આર્બિટ્રેશનની ફી અને ખર્ચની ચુકવણી કરીશું; અને (2) જો આર્બિટ્રેશનના અંતે આર્બિટ્રેશન તમને આર્બિટ્રેટરની નિમણૂંક કરવામાં આવે તે પહેલાં કરવામાં આવેલી અમારી છેલ્લી લેખિત ઑફર કરતાં વધુ ચૂકાદો આપે તો અમે તમને (i) એવોર્ડની રકમ અથવા $1,000 USD (બેમાંથી જે વધારે હોય તે) ચૂકવીશું; અને (ii) તમે લીધેલી કોઈ પણ વાજબી એટર્નીની ફી માટે; અને કોઈ પણ વાજબી ખર્ચ (નિષ્ણાત સાક્ષીની ફી અને ખર્ચ સહિત) જે તમારા વકીલે તમારી વ્યક્તિગત આર્બિટ્રેશનના સંબંધમાં મેળવ્યા હોય.

(f)         વિચ્છેદનીયતા

. જો તમામ અપીલો ખતમ થઈ ગયા પછી, જો અદાલતને આ અનુભાગ 2.14નો કોઈ પણ અનુભાગ અમલમાં મૂકી શકાય તેમ નથી કે કોઈ ઉપાય માટે કોઈ પણ દાવા કે વિનંતી અંગે કોઈ પણ દાવા કે વિનંતી કરવા માટે અદાલતમાં મુકદ્દમો કે ઉપાયો (જેમ કે જાહેર મનાઈહુકમના ઉપાય માટેની વિનંતી, આ કિસ્સામાં આર્બિટ્રેટર જવાબદારી અને વ્યક્તિગત રાહત અંગેનો ચુકાદો અદાલત સમક્ષ જારી કરે તે પહેલાં તે વિનંતીને ધ્યાનમાં લે છે). અન્યથા, જો અનુભાગ 2.14નો અન્ય કોઈ પણ અનુભાગ અમલમાં મૂકી શકાય તેમ ન હોવાનું જણાય, તો બાકીનો ભાગ અમલમાં રહેશે (કોઈ પણ અદાલતી કાર્યવાહી શરૂ થાય તે પહેલાં જારી કરવામાં આવેલા આર્બિટ્રેશન એવોર્ડ સાથે).

2.15.      સમગ્ર કરાર

. આ સમજૂતી (પ્રિન્ટેડ પેપર લાઇસન્સની શરતો અથવા અન્ય શરતો સાથે કોઈ પણ સૉફ્ટવેર સપ્લિમેન્ટ્સ, અપડેટ્સ અને સેવાઓ કે જે અમે પૂરી પાડી છે અને જેનો તમે ઉપયોગ કરો છો) સાથે મળીને, અને આ કરારમાં યાદીબદ્ધ વેબ લિંક્સમાં સમાવિષ્ટ શરતો, સૉફ્ટવેર અને આવા કોઈ પણ સપ્લિમેન્ટ્સ, અપડેટ્સ અને સેવાઓ માટેનો સંપૂર્ણ કરાર છે (સિવાય કે અમે આવા સપ્લિમેન્ટ્સ, સુધારાઓ, અથવા સેવાઓ સાથે અન્ય શરતો પ્રદાન કરીએ છીએ).

2.16.      વિવાદો ઉકેલવા માટે યોગ્ય કાયદો અને સ્થાન

. જો તમે યુનાઇટેડ સ્ટેટ્સ અથવા કેનેડામાં સૉફ્ટવેર મેળવ્યું હોય, તો તમે જ્યાં રહેતા હોય તે રાજ્ય અથવા પ્રાંતના કાયદાઓ (અથવા, જો કોઈ વ્યવસાય, જ્યાં તમારા મુખ્ય વ્યવસાયનું સ્થાન છે) આ કરારના અર્થઘટનને નિયંત્રિત કરે છે, તેના ભંગ માટેના દાવા, અને કાયદાના સિદ્ધાંતોના સંઘર્ષને ધ્યાનમાં લીધા સિવાય, અન્ય તમામ દાવા (ગ્રાહક સુરક્ષા સહિત, અયોગ્ય સ્પર્ધા અને ટોર્ટ્સના દાવાઓ), સિવાય કે FAA આર્બિટ્રેશન સાથે સંકળાયેલા તમામ બાબતોનું સંચાલન કરે છે. જો તમે અન્ય કોઇ દેશનું સૉફટવેર મેળવ્યું હોય, તો તેના કાયદાઓ લાગુ થાય છે, તેના સિવાય FAA આર્બિટ્રેશન સાથે સંકળાયેલી તમામ બાબતોને નિયંત્રિત કરે છે. જો યુ.એસ. ફેડરલ અધિકારક્ષેત્ર અસ્તિત્વમાં હોય, તો તમે અને Microsoft આર્બિટ્રેશન અથવા નાના દાવાની અદાલતમાં ન સાંભળવામાં આવ્યા હોય તેવા તમામ વિવાદો માટે કિંગ કાઉન્ટી, વોશિંગ્ટનની ફેડરલ કોર્ટમાં વિશિષ્ટ અધિકારક્ષેત્ર અને સ્થળ માટે સંમત થાઓ છો. જો તેમ ન હોય, તો તમે અને Microsoft આર્બિટ્રેશન અથવા નાના દાવાઓની અદાલતમાં ન સાંભળવામાં આવ્યા હોય તેવા તમામ વિવાદો માટે કિંગ કાઉન્ટી, વોશિંગ્ટનની સુપિરિયર કોર્ટમાં વિશિષ્ટ અધિકારક્ષેત્ર અને સ્થળ માટે સંમતિ આપો છો.

2.17.      ગ્રાહક અધિકારો; પ્રાદેશિક ભિન્નતા

. આ કરાર ચોક્કસ કાનૂની અધિકારો નું વર્ણન કરે છે. તમારા રાજ્ય, પ્રાંત અથવા દેશના કાયદા હેઠળ તમારી પાસે ગ્રાહક અધિકારો સહિત અન્ય અધિકારો હોઈ શકે છે. તમે સૉફ્ટવેર પ્રાપ્ત કર્યું છે તે પક્ષના સંદર્ભમાં પણ તમારી પાસે અધિકાર હોઈ શકે છે. જો તમારા રાજ્ય, પ્રાંત અથવા દેશના કાયદા આવું કરવા માટે મંજૂરી આપતા ન હોય તો આ કરાર તે અન્ય અધિકારોને બદલી શકતો નથી. ઉદાહરણ તરીકે, જો તમે નીચેના પ્રદેશોમાંથી કોઈ એકમાં સૉફ્ટવેર મેળવ્યું હોય, અથવા ફરજિયાત દેશનો કાયદો લાગુ પડે, તો નીચેની જોગવાઈઓ તમને લાગુ પડે છે:

(a)         ઑસ્ટ્રેલિયા

. ઑસ્ટ્રેલિયામાં રહેતા ગ્રાહકો માટે, અમારી સેવાઓ એવી બાંયધરીઓ સાથે આવે છે જેને ઓસ્ટ્રેલિયન કન્ઝ્યુમર લો હેઠળ બાકાત રાખી શકાય નહીં. સેવા સાથેની મોટી નિષ્ફળતાઓ માટે, તમે હકદાર છો:

(i)         અમારી સાથેનો તમારો સેવા સંબંધી કોન્ટ્રાક્ટ રદ કરવા માટે; અને

(ii)         બિનઉપયોગી ભાગ માટે રિફંડ માટે, અથવા તેના ઘટાડેલા મૂલ્ય માટે વળતર આપવા માટે.

તમે અન્ય કોઈ પણ વાજબી રીતે અપેક્ષિત નુકસાન અથવા નુકસાન માટે વળતર મેળવવા માટે પણ હકદાર છો. જો નિષ્ફળતા મોટી નિષ્ફળતા સમાન ન હોય તો તમે વાજબી સમયમાં સુધારેલી સેવામાં સમસ્યા હોવાનો હકદાર છો અને, જો આમ કરવામાં ન આવે તો, તમારો કરાર રદ કરવા અને કરારના ન વપરાયેલ ભાગ માટે રિફંડ મેળવવા માટે હકદાર છો.

(b)         જર્મની અને ઑસ્ટ્રિયા

. ઑસ્ટ્રેલિયામાં રહેતા ગ્રાહકો માટે, અમારી સેવાઓ એવી બાંયધરીઓ સાથે આવે છે જેને ઓસ્ટ્રેલિયન કન્ઝ્યુમર લો હેઠળ બાકાત રાખી શકાય નહીં. સેવા સાથેની મોટી નિષ્ફળતાઓ માટે, તમે હકદાર છો:

(i)         કોઈ કરારની ગેરંટી નથી . લાઇસન્સ પ્રાપ્ત સૉફ્ટવેરના સંબંધમાં Microsoft કોઈ કરારની બાંયધરી આપતું નથી.

(ii)         જવાબદારીની મર્યાદા . ઈરાદાપૂર્વકના વર્તન, કુલ બેદરકારી, ઉત્પાદન જવાબદારી અધિનિયમ (Product Liability Act) પર આધારિત દાવાઓના કિસ્સામાં, તેમજ મૃત્યુ અથવા વ્યક્તિગત અથવા શારીરિક ઈજાના કિસ્સામાં, Microsoft વૈધાનિક કાનૂન મુજબ જવાબદાર છે.

અગાઉના વિધાનને આધિન, Microsoft માત્ર ત્યારે જ થોડી બેદરકારી માટે જવાબદાર રહેશે જો Microsoft આવી ભૌતિક કરારની જવાબદારીઓનું ઉલ્લંઘન કરે છે, જેની પરિપૂર્ણતા આ કરારની યોગ્ય કામગીરીને સરળ બનાવે છે, જેનો ભંગ આ કરારના હેતુને જોખમમાં મૂકે છે અને જેના પાલનમાં પક્ષકાર સતત વિશ્વાસ કરી શકે છે (કહેવાતી "મુખ્ય જવાબદારીઓ"). સહેજ બેદરકારીના અન્ય કિસ્સાઓમાં, Microsoft સહેજ પણ બેદરકારી માટે જવાબદાર રહેશે નહીં.

2.18.      વૉરંટીનો અસ્વીકાર

. સોફ્ટવેર "જેમ છે તેમ", "તમામ ક્ષતિઓ સાથે" અને "ઉપલબ્ધ મુજબ" લાયસન્સ પ્રાપ્ત કરેલ છે. તમે તેનો ઉપયોગ કરવામાં થતુ જોખમ વેઠવા તૈયાર છો. MICROSOFT કોઈ સ્પષ્ટ વૉરંટી, બાંયધરીઓ અથવા ટર્મ્સ આપતું નથી. તમે તમારા સ્થાનિક કાયદા હેઠળ વધારાના ઉપભોક્તા અધિકારો અથવા વૈધાનિક બાંયધરી ધરાવતા હોઇ શકો છો જે આ કરારને બદલી શકતી નથી. તમારા સ્થાનિક કાયદાએ પરવાનગી આપેલ મર્યાદા સુધી, MICROSOFT તમામ લાગુ વૉરંટીને નિષ્કાસિત કરે છે, જેમાં વેપાર, કોઈ વિશેષ ઉદ્દેશ્ય માટે યોગ્યતા, અને બિન-ઉલ્લંઘન શામેલ છે.

2.19.      ઉપાયો અને નુકસાનીઓ પર મર્યાદા અને તેને બાકાત રાખવી

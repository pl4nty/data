# Invoke-HpEpscBootlogEvidenceValidation

## SYNOPSIS
Validates HP Endpoint Security Controller Bootlog Evidence.

## SYNTAX

```
Invoke-HpEpscBootlogEvidenceValidation [-HpEpscEvidence <Object[]>] [-HpEpscCerts <Object[]>] [-PlatformCertificate <String>] [-UseLocalPlatformCertificate] [-Nonce <Byte[]>] [-Uuid <String>] [-NoHpEpscCerts] [<CommonParameters>]
```

## DESCRIPTION

This cmdlet validates HP Endpoint Security Controller (EpSC) bootlog evidence. This includes evaluation of the security posture reported by the EpSC, the digital signature over that evidence generated by the reporting EpSC, and the certificate chain of the reporting EpSC that describes the key that was used by the reporting EpSC to sign the evidence. The cmdlet can also optionally confirm that the evidence includes a reference nonce and/or uuid value.

The cmdlet can operate in different modes:

1. **With EpSC Evidence and Certificates**: When both HpEpscEvidence and HpEpscCerts parameters are provided, the cmdlet performs the EpSC security posture validation, the digital signature over that evidence generated by the reporting EpSC, and the certificate chain of the reporting EpSC that describes the key that was used by the reporting EpSC to sign the evidence.

2. **Without EpSC Certificates**: When the NoHpEpscCerts switch is specified, the security posture validation is performed without validating the digital signature over the evidence that is generated by the reporting EpSC.

3. **Local vs External Platform Certificate**: The cmdlet can use either a local platform certificate (UseLocalPlatformCertificate) or an external one (PlatformCertificate parameter) to validate the reporting EpSC is the same component that was in the system when the HP Platform Certificate was issued by HP.

**DEPENDENCY**: HP Endpoint Security Controller Gen5 is required for retrieving evidence and certificates from the current platform.

## EXAMPLES

### Example 1
```powershell
PS C:\> $evidence = Get-HpEpscBootlogEvidence
PS C:\> $certs = Get-HpEpscCerts
PS C:\> Invoke-HpEpscBootlogEvidenceValidation -HpEpscEvidence $evidence -HpEpscCerts $certs -UseLocalPlatformCertificate
```

Validates EpSC bootlog evidence using certificates from the current system and the local platform certificate.

### Example 2
```powershell
PS C:\> $evidence = Get-HpEpscBootlogEvidence
PS C:\> $certs = Get-HpEpscCerts
PS C:\> Invoke-HpEpscBootlogEvidenceValidation -HpEpscEvidence $evidence -HpEpscCerts $certs -PlatformCertificate "C:\Certificates\HPInc.12345678.BASE.cer"
```

Validates EpSC bootlog evidence using certificates from the current system and a specific platform certificate file.

### Example 3
```powershell
PS C:\> $evidence = Get-HpEpscBootlogEvidence -Nonce $customNonce
PS C:\> $certs = Get-HpEpscCerts
PS C:\> Invoke-HpEpscBootlogEvidenceValidation -HpEpscEvidence $evidence -HpEpscCerts $certs -UseLocalPlatformCertificate -Nonce $customNonce
```

Validates EpSC bootlog evidence using a custom nonce for the validation process.

### Example 4
```powershell
PS C:\> $evidence = Get-HpEpscBootlogEvidence -Nonce $customNonce
PS C:\> $certs = Get-HpEpscCerts
PS C:\> Invoke-HpEpscBootlogEvidenceValidation -HpEpscEvidence $evidence -HpEpscCerts $certs -UseLocalPlatformCertificate -Nonce $customNonce -Uuid "12345678-1234-1234-1234-123456789abc"
```

Validates EpSC bootlog evidence using all possible evaluations including custom nonce and system UUID verification.

### Example 5
```powershell
PS C:\> $evidence = Get-HpEpscBootlogEvidence
PS C:\> Invoke-HpEpscBootlogEvidenceValidation -HpEpscEvidence $evidence -NoHpEpscCerts -UseLocalPlatformCertificate
```

Validates EpSC bootlog evidence without using HP EpSC certificates.

### Example 6
```powershell
PS C:\> $evidence = Get-HpEpscBootlogEvidence
PS C:\> $certs = Get-HpEpscCerts
PS C:\> Invoke-HpEpscBootlogEvidenceValidation -HpEpscEvidence $evidence -HpEpscCerts $certs -PlatformCertificate "C:\Certificates\" -Uuid "12345678-1234-1234-1234-123456789abc"
```

Validates EpSC bootlog evidence using a directory containing platform certificates and a specific system UUID.

### Example 7
```powershell
PS C:\> $evidence = Get-HpEpscBootlogEvidence
PS C:\> $certs = Get-HpEpscCerts
PS C:\> Invoke-HpEpscBootlogEvidenceValidation -HpEpscEvidence $evidence -HpEpscCerts $certs -PlatformCertificate "C:\Certificates\certificates.zip"
```

Validates EpSC bootlog evidence using platform certificates from a zip file.

## INPUTS

### None
This cmdlet does not accept pipeline input.

## OUTPUTS

### System.Object
Returns an object containing:
- Certificates: The HP EpSC certificates used for validation
- Evidence: The HP EpSC bootlog evidence that was validated

If no HP EpSC Certificates or Evidence are found, the cmdlet outputs an informational message.

## NOTES
- **DEPENDENCY**: HP Endpoint Security Controller Gen5 is required for retrieving evidence and certificates from the current platform
- This command requires an HP system with Endpoint Security Controller (EpSC) support
- The validation process verifies cryptographic signatures and attestation data integrity
- Nonce parameter should be a 32-byte array for proper validation
- When using NoHpEpscCerts parameter, some validation features may be limited
- Platform certificate validation is essential for establishing trust in the validation process
